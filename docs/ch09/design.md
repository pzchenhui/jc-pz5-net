# 9.3 添加并设计班主任助手的对话流

## 完整搭建流程

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">创建工作流</span>

1. 登录 Coze 平台
2. 点击左侧导航栏的 **"工作流"**
3. 点击 **"创建工作流"**
4. 命名为"学生评语生成"
5. 选择"**对话流**"模式
</div>

<div class="step-card">
<span class="step-number">2</span>
<span class="step-title">添加开始节点</span>

设置输入参数：
- `student_name`（学生姓名）：文本
- `grade_info`（学业表现）：文本
- `moral_info`（品德表现）：文本
- `strengths`（特长爱好）：文本（选填）
- `improvements`（需改进处）：文本（选填）
</div>

<div class="step-card">
<span class="step-number">3</span>
<span class="step-title">添加大模型节点</span>

这是核心处理节点，配置如下：

**系统提示词**：
```markdown
你是一位有10年经验的优秀班主任，擅长写温暖、
真诚、有个性的学生评语。

请根据以下信息，为该同学生成一段100-150字的学期评语：

学生姓名：{{student_name}}
学业表现：{{grade_info}}
品德表现：{{moral_info}}
特长爱好：{{strengths}}
需改进处：{{improvements}}

要求：
1. 用"你"来称呼学生，更有温度
2. 先肯定优点和亮点（2-3条）
3. 委婉地提出改进建议（1条）
4. 以鼓励和期望结尾
5. 语言亲切自然，不要太官方
6. 评语要个性化，避免模板化表述
```
</div>

<div class="step-card">
<span class="step-number">4</span>
<span class="step-title">添加结束节点</span>

将大模型节点的输出连接到结束节点，定义为工作流的最终输出。
</div>

## 📋 实操案例：评语生成效果

<div class="case-card">
<span class="case-label">📌 案例演示</span>

**输入信息**：
- 学生姓名：张小明
- 学业表现：成绩中上，数学和科学特别好，语文需要加强
- 品德表现：活泼热情，乐于助人，经常主动帮助同学
- 特长爱好：数学竞赛获奖，校篮球队主力
- 需改进处：粗心大意，作业经常漏题，书写潦草

**生成的评语**：

```
小明，这个学期你给了老师很多惊喜！你在数学和科学方面
展现了出色的天赋，还在数学竞赛中获奖，真为你感到骄傲。
更让老师感动的是你乐于助人的品质——你总是热情地帮助
同学，是班里公认的"热心肠"。球场上你那矫健的身姿也让
大家看到了你的活力。

如果你能在写作业时多一份细心，少一份急躁，把字写得再
工整一些，相信你会更加出色。新学期，老师期待看到一个
更加细心的你！加油！💪
```

</div>

## 对话流的测试清单

完成配置后，按以下清单测试：

| 测试项 | 方法 | 通过标准 |
|--------|------|---------|
| 正常流程 | 填写所有必填字段 | 正常生成评语 |
| 选填为空 | 不填选填字段 | 不报错，正常生成 |
| 边界值 | 输入超长文本 | 不报错 |
| 连续运行 | 连续生成3份评语 | 每次都正常 |
| 个性化 | 不同学生信息 | 评语风格不同 |

## 进阶：多评语风格切换

<div class="template-card">
<span class="template-label">📝 技巧</span>

可以在工作流中添加一个"风格选择"字段：

```
评语风格（下拉选择）：
- 温暖亲切型（适合低年级）
- 正式规范型（适合期末档案）
- 幽默活泼型（适合平时鼓励）
```

然后在大模型节点中根据选择的风格调整生成要求。

</div>
