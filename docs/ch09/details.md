# 9.4 其他细节设置：让班主任助手更贴心

班主任助手的细节设置直接影响使用体验。一个"有温度"的智能体，需要精心配置开场白、推荐问题、隐私保护等各项细节。

> ⏱️ **预计阅读用时**：约10分钟。

## 开场白设置

好的开场白让班主任愿意用、学生愿意问。

## 细节优化建议

### 1. 评语温度设置

班主任评语核心要求是**有温度、个性化**，不能千篇一律：

<div class="template-card">
<span class="template-label">📝 评语温度配置指南</span>

| 设置项 | 建议值 | 说明 | 配置方法 |
|--------|--------|------|---------|
| **语气风格** | 温暖鼓励型 | 避免负面评价，先肯定后建议 | 人设中预设语气："评语要有温度，像一位关心学生的老师在说话" |
| **个性化程度** | 高 | 每个学生评语不同，避免模板化 | 根据学生变量（性格、成绩、特长）生成差异化内容 |
| **字数控制** | 80-120字/人 | 简洁但有温度，不宜太长 | 人设中约束："每条评语控制在80-120字" |
| **结构模式** | 肯定→建议→鼓励 | 先表扬亮点，再委婉建议，最后鼓励 | 人设中预设结构："先肯定优点→委婉提出建议→以鼓励结尾" |
| **禁用词汇** | 差、笨、不行 | 不使用负面词汇 | 人设中约束："不使用负面评价词汇" |

### 评语结构示例

```
【肯定亮点】小明，你是一个聪明活泼的孩子，课堂上总能积极发言，特别是数学课上你的解题思路让老师印象深刻。

【委婉建议】如果能再多一些耐心，在做题时认真检查，相信你会取得更大的进步。

【鼓励结尾】老师期待看到一个更加认真细致的你！加油！💪
```

</div>

### 2. 隐私保护配置——重中之重

班主任助手涉及大量学生个人信息，**隐私保护是底线**。

<div class="case-card">
<span class="case-label">🔒 隐私安全配置方案</span>

### 必须遵守的4条红线

1. **不在知识库中存储学生个人信息**
   - 不上传学生姓名、家庭情况、成绩记录等到知识库
   - 学生信息通过对话流实时输入，用后不留存

2. **评语生成使用学号或代号而非姓名**
   - 输入："学号01，男生，成绩优秀，但注意力不集中"
   - 输出："该生学习能力强，思维活跃..."
   - 人工替换学号为真实姓名后再发布

3. **家校沟通话术不包含敏感信息**
   - 不输出学生家庭背景、经济状况等敏感信息
   - 话术模板使用占位符："[学生姓名]最近在[方面]有进步"

4. **定期清理对话历史**
   - 建议每周末清理一次对话历史
   - 平台设置对话保留期限为7天

### 隐私保护技术配置

| 配置项 | 设置值 | 说明 |
|--------|--------|------|
| 对话历史保留 | 7天 | 超过7天自动清除 |
| 知识库内容 | 仅评语词汇+话术模板 | 不含学生个人信息 |
| 数据导出 | 禁止 | 不允许导出对话记录 |
| 访问权限 | 仅班主任本人 | 团队空间中设置权限 |

</div>

### 3. 知识库配置

班主任助手的知识库应包含以下内容：

| 文件 | 用途 | 推荐数量 | 示例 |
|------|------|---------|------|
| **学生评语词汇库** | 丰富评语表达，避免重复 | 1份（200+条评价短语） | 优秀评语短语汇编 |
| **家校沟通话术模板** | 提供话术参考 | 1份（30+条话术） | 家校沟通话术手册 |
| **班会方案范例** | 参考格式和内容 | 2-3份 | 优秀班会方案汇编 |
| **班主任工作计划模板** | 计划撰写参考 | 1份 | 班主任学期工作计划模板 |
| **教育政策法规摘要** | 确保合规 | 1份 | 教育政策法规要点 |

### 4. 插件配置

| 插件 | 用途 | 推荐度 | 配置说明 |
|------|------|--------|---------|
| **文档处理** | 批量导出评语文档 | ⭐⭐⭐⭐⭐ | 生成Word格式评语，方便复制到成绩单 |
| **表格处理** | 整理班级数据 | ⭐⭐⭐⭐ | 分析成绩趋势、出勤率等 |
| **联网搜索** | 查找班会创意和活动案例 | ⭐⭐⭐ | 可选，用于获取最新活动灵感 |

## 多轮对话配置

班主任工作需要较多交互，建议配置：

| 设置项 | 建议值 | 说明 |
|--------|--------|------|
| **对话轮数** | 15-20轮 | 评语修改和家校沟通需要多轮交互 |
| **开启对话记忆** | ✅ 开启 | 支持方案逐步完善，不用重复说明背景 |
| **对话超时** | 30分钟 | 超过30分钟无操作自动重置，保护隐私 |
| **话题引导** | 开启 | 当用户偏离主题时，引导回班主任工作 |

## 安全红线

<div class="case-card">
<span class="exercise-label">⚠️ 班主任助手安全红线</span>

班主任助手**绝对不能**做的事情：

1. ❌ **存储或泄露学生个人信息**（姓名、家庭情况、成绩等）
2. ❌ **生成带有偏见的评语**（不因家庭背景、性别等区别对待）
3. ❌ **提供涉及学生隐私的沟通话术**（不讨论学生家庭背景）
4. ❌ **代替教师做教育判断**（如"这个学生应该转学"）
5. ❌ **输出不适合学生年龄的内容**

在人设中明确添加：
```
安全约束：
- 不存储、不输出学生个人敏感信息
- 评语必须公平、积极、有建设性
- 涉及学生问题时仅提供建议，最终判断权在教师
- 所有输出内容仅供教师参考，必须人工审核后使用
```

</div>
