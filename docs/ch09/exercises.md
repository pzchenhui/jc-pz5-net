### 李老师的班主任AI分身

李老师是某小学五年级班主任，管理40名学生。

**搭建前**：
1. 期末评语：40份×6分钟/份 = 4小时
2. 家校沟通：每次准备话术约30分钟，每周3-5次 ≈ 2小时
3. 班会方案：自己策划约1.5小时/次
4. 每周班主任文字和策划工作总耗时：**约10小时**

**搭建班主任助手后**：
1. 期末评语：AI批量生成→审核微调 = 40分钟
2. 家校沟通：AI提供话术模板 ≈ 20分钟/周
3. 班会方案：AI生成→调整 = 20分钟/次
4. 每周班主任文字和策划工作总耗时：**约2.5小时**

**量化效果**：节省约7.5小时/周，评语质量从"千篇一律"变为"个性化有温度"。

## 📋 练习二：对话流设计（25分钟）

<div class="exercise-card">
<span class="exercise-label">🔀 对话流设计模板</span>

### 参考设计：学生评语生成对话流

```
开始（班主任触发：生成功能评语）
  ↓
节点1：收集基本信息
  → 提问："请告诉我学生的学号和性别"
  → 输入变量：student_id, gender
  ↓
节点2：收集学业表现
  → 提问："该生本学期的学业表现如何？（优秀/良好/中等/需努力）"
  → 追问："有没有特别突出的学科或需要关注的学科？"
  → 输入变量：academic_level, special_subjects
  ↓
节点3：收集品德和特长
  → 提问："该生在品德和特长方面有什么突出表现？"
  → 追问："有没有参加什么活动或比赛？"
  → 输入变量：character, talents
  ↓
节点4：生成评语
  → 人设："你是一位温暖的班主任..."
  → 输入：以上所有变量
  → 输出：评语初稿
  ↓
节点5：确认和修改
  → 展示评语："这是生成的评语，您需要修改吗？"
  → 如果满意→结束
  → 如果需要修改→回到节点4，传入修改意见
  ↓
结束（输出最终评语）
```

## 💡 常见问题 FAQ

<div class="case-card">
<span class="exercise-label">❓ 常见问题</span>

**Q1：生成的评语太模板化怎么办？**
A：在人设中增加个性化约束："每条评语必须包含该学生的一个具体事例或特点，不能泛泛而谈"。

**Q2：评语语气太正式不够温暖怎么办？**
A：在人设中预设语气："像一位关心学生的老师在说话，使用温暖鼓励的语气，适当使用emoji"。

**Q3：如何确保不泄露学生隐私？**
A：知识库中不上传任何学生个人信息；人设中明确约束；定期清理对话历史。

**Q4：批量生成时评语太相似怎么办？**
A：逐个输入学生信息，确保每次输入的变量不同；在人设中强调"根据每个学生的特点差异化生成"。
