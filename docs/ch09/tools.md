# 9.6 巧用AI工具：解决班主任难题

## 班主任工作AI工具矩阵——全链路提效方案

班主任工作不只是写评语，还涉及**家校沟通、活动策划、数据分析、心理辅导、档案管理**等多个环节。AI工具可以覆盖班主任工作的全链路。

> ⏱️ **预计阅读用时**：约12分钟。

<div class="case-card">
<span class="exercise-label">📌 真实案例</span>

### 李老师的班主任AI办公实录

李老师是某小学五年级班主任，以下是她使用AI工具后的一周工作实录：

**周一**：
- 班主任助手（Coze） → 生成本周班会方案（15分钟）
- 人工调整 → 加入班级特色元素（5分钟）
- **总耗时：20分钟（传统方式约1.5小时）**

**周二**：
- 学生行为问题 → 班主任助手提供沟通话术（5分钟）
- 人工调整后与家长微信沟通（10分钟）
- **总耗时：15分钟（传统方式约30分钟）**

**周三**：
- 成绩数据整理 → Excel AI自动分析（10分钟）
- 班主任助手 → 生成学情分析简报（5分钟）
- **总耗时：15分钟（传统方式约1小时）**

**周四-周五**：
- 期末评语 → 班主任助手批量生成初稿（30分钟）
- 人工审核和微调（15分钟）
- **总耗时：45分钟（传统方式约4小时）**

> 💡 **一周班主任工作，AI辅助后从约15小时缩短到约3小时，节省约80%时间。**

## 分环节实操指南

### 环节一：评语生成——班主任助手智能体

<div class="case-card">
<span class="exercise-label">📝 评语生成工具实操</span>

**工具**：Coze班主任助手智能体
**适用场景**：期末评语、学期评语、操行评定

**操作流程**：
```
1. 打开班主任助手智能体
2. 选择"评语生成"功能
3. 输入学生信息（学号、性别、学科成绩、特长、行为表现）
4. AI生成评语初稿
5. 人工审核和微调
6. 批量导出到成绩单
```

**各环节耗时对比**：

| 班级人数 | 传统方式 | AI辅助 | 节省 |
|---------|---------|--------|------|
| 30人 | 3小时 | 25分钟 | 86% |
| 40人 | 4小时 | 40分钟 | 83% |
| 50人 | 5小时 | 50分钟 | 83% |

### 环节二：家校沟通——话术建议工具

<div class="case-card">
<span class="exercise-label">💬 家校沟通工具实操</span>

**工具**：班主任助手 / 智谱清言
**适用场景**：成绩下降通知、行为问题沟通、表扬反馈

**提示词模板**：
```
我班上有一个学生最近数学成绩从85分下降到62分，
我需要跟家长沟通这件事。

请帮我准备：
1. 沟通话术（微信文字版）
2. 面谈建议（如果家长来学校）
3. 需要收集的学生信息清单
4. 后续跟进建议

要求：
- 语气委婉，不制造焦虑
- 先肯定学生的优点
- 客观描述成绩变化
- 提出具体可行的建议
```

**输出示例**：
```
📱 微信沟通话术：

XX家长您好！最近孩子在课堂上表现很好，
特别热心帮助同学。不过我注意到数学成绩
有些波动，从之前的85分降到了62分。

我想了解一下孩子最近在家里的学习情况，
看看是不是遇到了什么困难。如果您方便的话，
我们可以约个时间当面聊聊，一起帮助孩子。

您看什么时间方便呢？😊
```

<div class="comparison-card">
<span class="comparison-label">📊 班主任难题→AI解法对照表</span>

| **数据汇总麻烦** | 手工统计+制表 | AI自动整理分析+图表 | 2小时→15分钟 | 更准确 |
| **成长档案整理** | 每学期手动整理 | AI定期汇总+格式化 | 3小时→20分钟 | 更规范 |

## 实操案例：期末评语批量生成全流程

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">准备学生信息</span>

准备每个学生的基本信息表格（学号、性别、学科成绩、特长、行为表现关键词）。

<span class="step-number">2</span>
<span class="step-title">逐个输入信息</span>

在班主任助手中逐个输入学生信息，AI自动生成评语初稿。

<span class="step-number">3</span>
<span class="step-title">批量审核</span>

通览所有评语，重点检查：是否有模板化、是否公平、是否有温度、是否有误。

<span class="step-number">4</span>
<span class="step-title">微调完善</span>

对需要调整的评语进行微调，确保每个学生都有个性化的评价。

<span class="step-number">5</span>
<span class="step-title">导出使用</span>

将最终版评语导出或复制到成绩报告单。
</div>

> 以40人的班级为例，整个过程约需 **40分钟**，传统方式需要 **4-6小时**。
