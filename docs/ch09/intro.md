## 什么是学校AI工作台？

学校AI工作台是一个**统一的AI服务入口**，将各类智能体整合在一起，让师生按需使用。

```
┌──────────────────────────────────────────┐
│            学校AI工作台                    │
│                                          │
│  ┌─────────┬─────────┬──────────┐        │
│  │ 教学区  │ 管理区  │  学习区   │        │
│  ├─────────┼─────────┼──────────┤        │
│  │备课助手 │行政助手 │智慧学伴  │        │
│  │教学设计 │活动助手 │学科问答  │        │
│  │测评助手 │数据助手 │心理辅导  │        │
│  └─────────┴─────────┴──────────┘        │
│                                          │
│  ┌──────────────────────────────────┐    │
│  │  使用数据统计 · 反馈收集 · 持续迭代 │    │
│  └──────────────────────────────────┘    │
└──────────────────────────────────────────┘
```

## 搭建方案对比

<div class="comparison-table">

<div class="comparison-card good">

<span class="template-label">✅ 方案一：Coze 团队空间（推荐起点）</span>

**搭建步骤**：
1. 在 Coze 创建团队/企业空间
2. 将各类智能体发布到团队空间
3. 按角色分配访问权限
4. 生成工作台页面链接
5. 通过学校工作群分享

**优点**：
- ✅ 零技术门槛
- ✅ 快速上线（1-2天）
- ✅ 免费额度可用
- ✅ 平台维护，无需技术投入

**缺点**：
- ⚠️ 功能受平台限制
- ⚠️ 数据在第三方平台

**适合**：刚开始探索AI的学校

**成本**：免费 - 数百元/月（按使用量）

## 权限管理方案

| 角色 | 可访问内容 | 管理权限 |
|------|-----------|---------|
| **校长/管理员** | 全部智能体 + 使用数据 | 用户管理、权限分配、数据查看 |
| **中层干部** | 教学类 + 管理类智能体 | 本部门智能体管理 |
| **学科教师** | 备课、测评、教研类智能体 | 个人智能体管理 |
| **班主任** | 班主任、活动、家校沟通类 | 班级相关智能体 |
| **学生** | 智慧学伴、学科问答 | 仅使用 |
| **家长** | 家校沟通助手 | 仅使用 |

## 📋 推广路线图

### 阶段一：试点期（1-2个月）

| 任务 | 负责人 | 产出 |
|------|--------|------|
| 选3-5个骨干教师试用 | 教务处 | 试用反馈报告 |
| 搭建2-3个核心智能体 | 试点教师 | 可用的智能体 |
| 收集使用数据和问题 | 试点教师 | 问题清单 |

### 阶段二：推广期（2-3个月）

| 任务 | 负责人 | 产出 |
|------|--------|------|
| 教研组长培训 | 试点教师 | 各教研组至少1人会用 |
| 完善核心智能体 | 试点教师 | 质量提升的智能体 |
| 编写使用手册 | 教务处 | 《学校AI使用指南》 |

### 阶段三：普及期（3-6个月）

| 任务 | 负责人 | 产出 |
|------|--------|------|
| 全校教师培训 | 教务处 | 80%以上教师会用 |
| 纳入教师发展计划 | 校长室 | 制度化推进 |
| 建立AI教研机制 | 教研组 | 常态化交流 |

### 阶段四：深化期（持续）

| 任务 | 负责人 | 产出 |
|------|--------|------|
| 持续迭代智能体 | 全体教师 | 越来越好用 |
| 形成校本特色 | 教研组 | 校本AI资源库 |
| 对外展示交流 | 校长室 | 品牌影响力 |

## 使用数据追踪

| 指标 | 说明 | 获取方式 |
|------|------|---------|
