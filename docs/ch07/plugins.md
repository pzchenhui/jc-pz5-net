# 7.3 行政助手插件的选择与配置

## 行政助手需要的插件

| 插件 | 用途 | 重要程度 | 行政场景 |
|------|------|---------|---------|
| 联网搜索 | 查询最新政策、时事、法规 | ⭐⭐⭐⭐⭐ | 写公文时引用最新政策 |
| 文档阅读 | 读取上传的文档进行整理 | ⭐⭐⭐⭐ | 阅读长文件后提取要点 |
| 代码执行 | 数据统计与计算 | ⭐⭐⭐ | 成绩统计、数据分析 |
| 日历 | 会议安排、日程管理 | ⭐⭐ | 自动查询日期信息 |

## 📋 实操步骤

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">添加联网搜索插件</span>

这是行政助手最重要的插件——写公文时经常需要引用最新政策。

1. 在插件市场搜索"联网搜索"或"搜索"
2. 添加搜索结果插件
3. 测试：输入"查询最新的义务教育课程方案"
</div>

<div class="step-card">
<span class="step-number">2</span>
<span class="step-title">添加文档阅读插件</span>

用于读取上传的长文档后提取要点。

1. 添加文档阅读插件
2. 上传一份长文件（如工作总结素材）
3. 测试：输入"请阅读这份文件，提取主要工作内容"
</div>

<div class="step-card">
<span class="step-number">3</span>
<span class="step-title">配置知识库</span>

| 文件 | 用途 |
|------|------|
| 学校简介和基本信息 | 提供学校基础数据（人数、班级数等） |
| 公文格式规范 | 确保格式正确 |
| 往年优秀公文范例 | 参考风格和格式 |
| 政策法规摘要 | 常用政策引用 |
| 学校管理制度汇编 | 确保内容合规 |
</div>

## 使用技巧

### 技巧一：引用最新政策

```
请帮我搜索2024年最新的"双减"政策要求，
然后写一份学校落实"双减"的工作总结。
```

### 技巧二：读取文件后整理

```
请阅读我上传的各部门工作总结，
汇总成一份学校年度工作总结，
要求：有数据、有亮点、有不足、有展望。
```
