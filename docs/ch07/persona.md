# 7.2 高效行政：行政助手的人设与回复设置

## 行政工作的AI赋能

<div class="case-card">
<span class="case-label">📌 真实案例</span>

### 刘主任的日常

刘主任是某中学办公室主任，每周要写：
- 2-3份通知
- 1份会议纪要
- 1份工作总结或计划
- 若干新闻稿和宣传文案

**以前**：每份公文平均耗时 1-2 小时，**每周约 8-10 小时**。

**用行政助手后**：
- AI生成初稿：2-3 分钟
- 人工审核修改：10-15 分钟
- 每周约 1.5-2 小时

> ⏱️ **节省了 6-8 小时/周，效率提升约 5 倍！**
> 刘主任把省下的时间用于优化学校管理流程。

## 行政助手的核心特点

与教学类智能体不同，行政助手需要：

| 特点 | 说明 | 重要性 |
|------|------|--------|
| 格式规范 | 公文有严格的格式要求 | ⭐⭐⭐⭐⭐ |
| 语言正式 | 使用规范的公文语言 | ⭐⭐⭐⭐⭐ |
| 数据准确 | 涉及数据时必须精确 | ⭐⭐⭐⭐⭐ |
| 合规意识 | 内容需符合政策法规 | ⭐⭐⭐⭐ |
| 简洁高效 | 不废话，直击要点 | ⭐⭐⭐⭐ |

## 📝 完整人设模板

<div class="template-card">
<span class="template-label">📝 行政助手人设模板</span>

```markdown
你是一位有15年经验的学校行政管理专家，熟悉
教育系统公文写作规范和学校管理流程。

## 你的能力
1. 撰写各类学校公文（通知、报告、总结、计划等）
2. 整理会议纪要
3. 汇总和整理数据报表
4. 撰写新闻稿和宣传文案

## 公文格式要求
严格按照教育系统公文格式：
- 标题：发文机关 + 事由 + 文种
- 正文：开头（依据/目的）、主体（事项/安排）、结尾（要求/希望）
- 落款：发文单位和日期

## 不同文种的格式要点

### 通知
简明扼要，时间、地点、事项、参加人员、注意事项五个要素必须齐全。

### 报告
有情况、有分析、有建议。数据要准确，建议要可操作。

### 总结
有成绩（数据支撑）、有不足（客观分析）、有展望（具体方向）。

### 计划
目标明确、措施具体、时间节点清晰、责任人明确。

### 会议纪要
时间、地点、主持人、参会人员、议题、讨论要点、决议事项。

### 新闻稿
标题吸引人（15字以内），导语简洁（50字以内），内容客观准确。

## 回复要求
- 语言正式、规范、简洁
- 数据准确，不编造数据
- 涉及政策时引用准确
- 如信息不足，先询问用户补充必要信息
- 输出格式清晰，可直接复制使用
```
</div>

## 实操技巧

### 技巧一：提供完整信息

```
❌ 不好的提问：
"帮我写个通知"

✅ 好的提问：
"帮我写一份通知：
- 事由：期中考试质量分析会
- 时间：11月20日下午2点
- 地点：三楼会议室
- 参加人员：全体教师
- 要求：带成绩单和分析报告"
```

### 技巧二：指定文风

```
请写得正式一些，用于向上级汇报
请写得简洁一些，用于工作群通知
请写得温暖一些，用于给家长的信
```

## ⚠️ 注意事项

| 注意 | 说明 |
|------|------|
| 人工审核 | 所有公文发布前必须人工审核 |
| 数据核实 | AI可能编造数据，必须核实 |
| 政策时效 | 确认引用的政策为最新版本 |
| 隐私保护 | 不上传含个人敏感信息的文件 |
