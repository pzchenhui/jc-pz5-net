# 7.8 巧用AI工具：搞定各类校园活动

## 活动全流程——AI工具矩阵

校园活动不只是策划，还包括**宣传、执行、复盘**等多个环节。

## 活动全流程AI工具

| 环节 | AI工具 | 具体应用 | 推荐平台 |
|------|--------|---------|---------|
| 策划 | 方案生成 | 活动方案+流程+分工 | Coze |
| 宣传 | 海报文案+新闻稿 | 海报设计建议+新闻稿 | 智谱清言 |
| 执行 | 现场主持稿 | 主持词+串词 | Coze |
| 复盘 | 数据分析+总结 | 参与度统计+改进建议 | Kimi |
| 归档 | 资料整理 | 活动资料分类整理 | Kimi |

## 各活动类型最佳实践

| 活动类型 | 推荐工作流 | 推荐平台 |
|----------|-----------|---------|
| 运动会 | 策划→海报→主持稿→复盘 | Coze |
| 文艺汇演 | 策划→海报→新闻稿→复盘 | Coze+智谱清言 |
| 主题班会 | 方案→主持稿→总结 | Coze |
| 毕业典礼 | 流程→致辞稿→新闻稿 | Coze+智谱清言 |
| 科技节 | 方案→海报→新闻稿→复盘 | Coze+Kimi |
