## 📌 参考答案与评判标准

### 活动方案评判标准

<div class="case-card">
<span class="case-label">📋 参考答案</span>

**优秀活动方案应包含的要素**：

1. **活动主题**：有吸引力、符合活动性质、适合目标群体
2. **活动时间地点**：具体明确，考虑天气备选方案
3. **活动流程**：时间轴形式，每个环节标注负责人
4. **人员分工**：覆盖所有岗位（主持、裁判、后勤、安保、医疗）
5. **物资清单**：分类列出（器材、道具、奖品、文具、药品）
6. **预算明细**：每项有单价和总价，总额不超预算
7. **安全预案**：至少包含3种应急场景（天气突变、人员受伤、设备故障）

**合格线**：上述7要素齐全，方案可直接作为讨论基础。

</div>

### 海报文案评判标准

**优秀海报文案标准**：
1. 标题醒目（10字以内，有吸引力）
2. 信息完整（时间、地点、参与方式齐全）
3. 有号召力（口号朗朗上口）
4. 风格独特（不与其他活动混淆）
5. 设计建议合理（配色/排版/配图符合活动性质）

**合格线**：5要素齐全，至少有1个版本可直接使用（微调后）。

### 刘老师的科技节全流程实践

刘老师是某初中科技节负责人，往年独自策划，今年使用AI辅助。

**AI辅助全流程**：
- 方案策划：输入"科技节、1200人、预算15000元、4个学科" → 30分钟生成3套方案
- 海报文案：生成5个版本，选择最优 → 10分钟
- 新闻稿：活动后输入现场数据 → 15分钟生成800字新闻稿
- 复盘报告：上传参与数据+反馈问卷 → 20分钟生成复盘报告

**效果对比**：
- 2024年手动操作：约3天（方案）+ 1天（文案）+ 2小时（新闻稿）+ 2小时（复盘）= **约4.5天**
- 2025年AI辅助：约1小时（方案）+ 15分钟（文案）+ 20分钟（新闻稿）+ 25分钟（复盘）= **约2小时**
- **效率提升约97%**

刘老师说："AI生成的方案虽然不是最终版，但它给了我们一个很好的讨论基础。以前我们讨论从'写什么'开始，现在从'怎么改'开始，效率完全不同。"

### 李老师的文艺汇演策划

**AI辅助过程**：
