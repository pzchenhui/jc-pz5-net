# 📝 第7章练习：行政助手

搭建行政助手智能体，能撰写公文、整理会议纪要、汇总数据。

## 🛠️ 实操任务：搭建行政助手

### 步骤检查清单

| 步骤 | 完成标准 | 打勾 |
|------|---------|------|
| 1. 编写行政人设 | 包含公文格式要求 | □ |

| 3. 测试通知生成 | 格式规范、要素齐全 | □ |

## 📋 练习一：公文格式审核（15分钟）

**生成一份通知，检查格式是否规范：**

| 检查项 | 是否包含 | 说明 |
|--------|---------|------|
| 标题完整 | □是 □否 | 发文机关+事由+文种 |
| 称呼正确 | □是 □否 | 如"各班级：" |
| 正文结构 | □是 □否 | 开头、主体、结尾 |
| 时间地点 | □是 □否 | 事项明确 |
| 落款日期 | □是 □否 | 单位和日期 |
