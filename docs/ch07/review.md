# 7.5 活动复盘与总结智能体

## 活动复盘——从经验中成长

活动复盘是提升活动质量的关键环节，但往往**被忽视或流于形式**。

<div class="case-card">
<span class="case-label">📌 真实案例</span>

### 文艺汇演复盘

**传统方式**：
1. 回忆活动流程→记录亮点和问题→写总结报告
2. 耗时：约2小时
3. 容易遗漏细节

**使用AI智能体后**：
1. 输入活动信息和现场记录→AI按模板整理→生成复盘报告
2. 耗时：约20分钟

> ⏱️ **节省了约1.5小时，效率提升约6倍。而且复盘更全面。**

</div>

## 复盘智能体搭建

### 人设模板

<div class="template-card">
<span class="template-label">📝 活动复盘人设</span>

```
你是一位校园活动复盘专家，具备以下能力：

1. 能够从活动数据中提取关键信息
2. 能够客观分析活动的成功因素和不足之处
3. 能够提供可操作的改进建议
4. 能够生成结构化的复盘报告

复盘维度：
- 参与度（参与人数、参与率、活跃度）
- 满意度（参与者反馈评分）
- 执行度（流程执行情况、人员配合）
- 安全度（安全事件、应急处理）
- 影响力（活动影响范围、媒体报道）

输出格式：
- 活动概览
- 数据分析
- 亮点总结
- 问题与建议
- 下次改进方向
```

</div>

### 典型输出

```
📊 活动复盘报告：2025年春季运动会

一、活动概览
- 活动时间：2025年4月10日
- 参与人数：850人（参与率92%）
- 活动时长：4小时

二、数据分析
- 参与度：⭐⭐⭐⭐⭐（92%）
- 满意度：⭐⭐⭐⭐（4.2/5.0）
- 执行度：⭐⭐⭐⭐（流程完成度95%）
- 安全度：⭐⭐⭐⭐⭐（零事故）

三、亮点总结
1. 开幕式表演精彩，获得一致好评
2. 项目设置合理，各年级都有参与
3. 后勤保障到位

四、问题与建议
1. 问题：部分项目等待时间过长
   建议：下次增加并行项目，减少等待
2. 问题：奖品发放不够及时
   建议：提前分组，安排专人负责

五、下次改进方向
1. 优化项目时间安排
2. 增加观众互动环节
3. 改进奖品发放流程
```
