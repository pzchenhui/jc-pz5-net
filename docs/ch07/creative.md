# 7.3 活动创意策划智能体

## 活动策划——从创意到方案

活动创意策划是活动助手最核心的功能：

<div class="case-card">
<span class="case-label">📌 真实案例</span>

### 刘老师的科技节策划

刘老师是某初中科技节负责人，往年都是自己策划，今年尝试用AI辅助。

**传统方式**：
1. 查资料→想主题→设计方案→列流程→排人员
2. 耗时：约3-4天

**使用AI活动智能体后**：
1. 告诉AI活动类型和预算→AI生成3个方案→选择最优→微调
2. 耗时：约1小时

> ⏱️ **节省了约3天，效率提升约10倍。**

</div>

## 策划工作流搭建

<div class="case-card">
<span class="case-label">📝 活动策划工作流</span>

```
开始（活动类型+参与对象+预算）
  ↓
节点1：创意生成
  → 生成3个不同风格的活动主题
  → 每个主题配简短描述
  ↓
节点2：方案详细化
  → 选择最优主题
  → 生成详细方案：
    - 活动流程
    - 人员分工
    - 物资清单
    - 预算明细
    - 安全预案
  ↓
节点3：海报文案
  → 生成活动海报文案
  → 提供设计建议
  ↓
节点4：输出整理
  → 整理成统一格式
  → 生成活动方案文档
  ↓
结束
```

</div>
