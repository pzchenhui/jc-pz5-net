# 7.6 其他细节设置：让活动助手更贴心

## 开场问候语定制：让活动助手"专业又好用"

活动助手的开场白需要传达两个信息：一是它的能力范围，二是它的专业感。

### 开场问候语的关键作用

1. **明确能力边界**：告诉用户助手能策划哪些类型的活动
2. **建立专业信任**：让用户相信助手能提供有价值的方案
3. **引导有效输入**：引导用户提供活动关键信息（类型/规模/预算）
4. **降低使用门槛**：通过推荐问题让用户快速开始

<div class="template-card">
<span class="template-label">📝 推荐开场白模板</span>

### 标准版

```
你好！我是校园活动助手 🎉

我可以帮你：
📋 策划活动方案（运动会/文艺汇演/主题班会等）
🎨 生成海报文案和配图建议
📝 撰写活动新闻稿
📊 活动复盘与总结

告诉我你要办什么活动吧！
```

### 专业版

```
你好！我是学校活动策划助手 📋

我能帮你完成从策划到复盘的全流程：
1. 🎯 活动方案策划（含时间线、人员分工、预算）
2. 📢 宣传文案撰写（海报、通知、新闻稿）
3. 📊 活动复盘分析（亮点、不足、改进建议）

请告诉我：你要办什么活动？时间、规模、预算大概是多少？
```

### 简洁版

```
校园活动助手上线 🎯

活动策划 / 宣传文案 / 新闻稿 / 复盘总结
告诉我你的需求，我来帮你搞定！
```

## 首条消息设置：用户开口后的第一个回复

首条消息决定了用户对助手专业能力的第一判断。

<div class="template-card">
<span class="template-label">📝 首条回复模板</span>

**收到活动需求后的首条回复**：
```
好的！我来帮你策划这个活动。📋

为了给出最合适的方案，我想了解几个关键信息：
1. 活动时间和地点确定了吗？
2. 预计参加人数大概多少？
3. 预算有没有大概的范围？
4. 有没有特别的要求或限制？

你先告诉我，我来帮你设计方案！
```

**用户给出完整信息后的首条回复**：
```
好的，信息收到了！我来帮你策划。

根据你的需求，我建议从以下几个方面着手：
1. 活动方案（含时间线和人员分工）
2. 宣传方案（海报/通知/新闻稿）
3. 应急预案（安全/天气/人员）

我先给你出一个方案框架，你看看哪里需要调整？
```

## 知识库配置建议

活动助手的知识库需要包含活动方案模板、往年案例、学校规范等信息。

<div class="template-card">
<span class="template-label">📝 知识库配置方案</span>

| 文件类型 | 内容 | 推荐数量 | 说明 |
|----------|------|---------|------|
| **往年活动方案** | 学校过去的活动方案 | 2-5 份 | 让助手了解学校风格和习惯 |
| **场地信息** | 学校可用场地条件 | 1 份 | 操场/礼堂/会议室的容量和设备 |
| **安全管理制度** | 学校安全规范 | 1 份 | 确保方案符合安全要求 |
| **经费预算标准** | 各类活动的预算参考 | 1 份 | 帮助控制预算在合理范围 |
| **学校简介** | 学校基本信息 | 1 份 | 校名/校训/办学理念等 |
| **活动模板** | 活动方案标准模板 | 3-5 份 | 运动会/汇演/班会的标准格式 |
| **新闻稿范文** | 过去的优秀新闻稿 | 5-10 篇 | 供助手参考写作风格 |

### 知识库上传步骤

1. **收集资料**：整理往年活动方案、学校制度、场地信息等
2. **格式统一**：转为 PDF 或 Word 格式
3. **上传到平台**：在智能体的"知识库"设置中上传
4. **分类管理**：按文件类型命名，方便后续维护
5. **测试检索**：输入"运动会方案"，检查是否能检索到相关内容

| 插件 | 用途 | 推荐度 | 说明 |
|------|------|--------|------|

```
用户输入活动需求

图片生成 → 制作宣传海报
    ↓
文档处理 → 导出为 Word 文档
    ↓
输出完整方案包
```

## 多轮对话配置

多轮对话让助手能支持活动方案的逐步完善。

| 设置项 | 建议值 | 说明 |
|--------|--------|------|
| 对话轮数 | 20-30 轮 | 策划活动需要较多交互，建议设高一些 |
| 开启对话记忆 | ✅ 开启 | 支持方案逐步完善，记住之前的讨论内容 |
| 话题引导 | 开启 | 引导用户补充关键信息（时间/地点/人数/预算） |
| 超时重置 | 60 分钟 | 活动策划讨论时间较长，超时建议设 60 分钟 |

### 多轮对话工作流示例

<div class="case-card">
<span class="case-label">📝 活动策划多轮对话示例</span>

**场景**：策划春季运动会

```
第 1 轮：
用户：帮我策划一场春季运动会
助手：好的！我来帮你策划春季运动会。🏃‍♂️

为了给出最合适的方案，请告诉我：
1. 预计什么时间举办？
2. 大概多少学生参加？
3. 预算有没有范围？
4. 有没有特别想加入的项目？

第 2 轮：
用户：4 月中旬，全校 800 人参加，预算 5000 元
助手：好的，信息收到了！

根据你们的情况，我建议的运动会方案：
📅 时间：4 月中旬（具体日期待定）
👥 规模：800 人，预计 2 天
💰 预算：5000 元

初步方案框架：
第一天：开幕式 + 田赛项目
第二天：径赛项目 + 趣味项目 + 闭幕式

你看看这个框架，需要调整的地方告诉我？

第 3 轮：
用户：趣味项目可以多一些吗？学生们最喜欢
助手：当然可以！趣味项目能提高学生参与度。👍

推荐的趣味项目：
1. 接力运球（每班 10 人）
2. 拔河比赛（按年级分组）
3. 三人四足（师生组队）
4. 袋鼠跳（低年级专属）
5. 障碍跑（综合体能）

你觉得哪些适合加入？我可以调整方案。

第 4 轮：
用户：都要！再帮我把预算分配也列出来
助手：好的！我来更新预算分配：

📊 预算明细（总计 5000 元）：
- 奖品（奖杯/证书/小礼品）：1500 元
- 器材（趣味项目道具）：800 元
- 宣传（海报/横幅）：500 元
- 后勤（水/急救包）：700 元
- 开幕式（音响/道具）：1000 元
- 预留应急：500 元

方案还需要调整吗？
```

## 操作截图描述

以下是关键设置页面的操作指引：

<div class="case-card">
<span class="case-label">📸 设置页面指引</span>

### 开场白设置

**操作路径**：智能体设置 → 对话设置 → 开场白

**设置步骤**：
1. 点击"编辑开场白"
2. 输入上文提供的开场白模板（选择适合你场景的版本）
3. 点击"保存"
4. 测试效果：新建对话查看开场白是否正常显示

**注意**：开场白不要太长，5-6 行最佳。关键是要让用户一眼看懂这个助手能做什么。

### 推荐问题设置

**操作路径**：智能体设置 → 对话设置 → 推荐问题

**设置步骤**：
1. 点击"添加推荐问题"
2. 输入 5-8 个用户最常问的问题
3. 按使用频率排序（最常问的排前面）
4. 点击"保存"
5. 测试效果：查看推荐问题是否显示在对话框下方

### 知识库配置

**操作路径**：智能体设置 → 知识库 → 上传文件

**设置步骤**：
1. 点击"上传文件"
2. 选择准备好的资料（往年方案/场地信息/安全制度等）
3. 等待文件处理完成
4. 测试检索：在对话框中输入"运动会"，检查是否能检索到相关内容

### 插件配置

**操作路径**：智能体设置 → 插件 → 添加插件

**设置步骤**：
1. 点击"添加插件"
2. 选择需要的插件（联网搜索/图片生成/文档处理等）
3. 配置插件参数（如联网搜索的搜索范围限制）
4. 点击"保存"
5. 测试效果：在对话中测试插件功能是否正常

### 对话记忆设置

**操作路径**：智能体设置 → 高级设置 → 对话记忆

**设置步骤**：
1. 开启"对话记忆"
2. 设置对话轮数上限（活动策划建议 20-30 轮）
3. 设置超时时间（建议 60 分钟）
4. 点击"保存"

</div>

## 不同活动类型的配置差异

不同活动类型需要不同的配置重点：

| 活动类型 | 知识库重点 | 插件重点 | 对话轮数 | 安全等级 |
|----------|-----------|---------|---------|---------|
| **运动会** | 往届方案/场地/安全制度 | 计算器（预算） | 20-30 | 高（安全要求多） |
| **文艺汇演** | 节目模板/主持人稿范文 | 图片生成（海报） | 25-30 | 中 |
| **主题班会** | 班会方案模板/教育素材 | 文档处理 | 15-20 | 低 |
| **毕业典礼** | 流程模板/发言稿范文 | 图片生成（邀请函） | 20-25 | 中 |
| **科技节** | 活动方案/科学实验案例 | 联网搜索（新创意） | 20-25 | 高（实验安全） |
