## 细节优化建议

### 1. 公文模板预设

在人设中预置常用公文模板（参考7.2节的模板库），包括：

| 模板类型 | 适用场景 | 关键要素 |
|---------|---------|---------|
| 通知模板 | 会议通知、活动通知 | 时间、地点、对象、事项 |
| 报告模板 | 工作总结、情况汇报 | 背景、过程、结果、建议 |
| 请示模板 | 经费申请、活动申请 | 事由、方案、预算、请示 |
| 新闻稿模板 | 活动报道、成果宣传 | 时间、事件、意义、展望 |

### 2. 知识库配置

| 文件 | 用途 | 推荐数量 |
|------|------|---------|
| 学校简介和基本信息 | 提供学校基础数据 | 1份 |
| 公文格式规范文件 | 确保格式正确 | 1-2份 |
| 往年优秀公文范例 | 参考风格和格式 | 3-5份 |
| 政策法规摘要 | 常用政策引用 | 1份 |
| 学校管理制度汇编 | 确保内容合规 | 1份 |

### 3. 插件配置

| 插件 | 用途 | 推荐度 |
|------|------|--------|
| 联网搜索 | 查找最新政策法规 | ⭐⭐⭐⭐ |
| 文档处理 | 生成Word/PDF文档 | ⭐⭐⭐⭐⭐ |

### 4. 多轮对话配置

| 设置项 | 建议值 | 说明 |
|--------|--------|------|
| 对话轮数 | 10-15轮 | 公文撰写需要收集关键信息 |
| 开启对话记忆 | ✅ 开启 | 支持公文逐步修改完善 |

## 安全红线

<div class="case-card">
<span class="case-label">⚠️ 行政安全注意事项</span>

1. **涉密文件不上传**：人事变动、财务数据等敏感内容不交给AI
2. **最终审核权在人**：所有公文发布前必须人工审核
3. **数据脱敏处理**：学生姓名、身份证号等替换为占位符
4. **保留原始版本**：AI生成后保留人工修改记录
5. **定期更新知识库**：政策法规变化时及时更新

</div>
