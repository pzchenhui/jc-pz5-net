# 1.3 教师如何有效利用AI智能体？

## 从"会用"到"用好"的四个层次

```
Level 1：会用工具 → Level 2：搭建智能体 → Level 3：编排工作流 → Level 4：构建生态
```

<div class="case-card">
<span class="case-label">📌 真实成长路径</span>

### 张老师的AI成长记

**第1周（Level 1）**：
- 用 DeepSeek 写了一封家长会通知
- 感觉"挺好用的"，但每次都从头输入

**第2周（Level 2）**：
- 搭建了第一个智能体"通知写作助手"
- 设定了学校的格式要求
- 发现每次都能生成符合学校规范的文案

**第1个月（Level 3）**：
- 搭建了"备课工作流"：查课标→生成教案→出题→做课件
- 一键完成整套备课流程

**第3个月（Level 4）**：
- 在教研组推广，5位老师都建了自己的智能体
- 建立了学科智能体共享库

> 张老师说："从一个人用到全组用，AI的价值放大了10倍。"

### 各层次详细说明

| 层次 | 能力 | 时间建议 | 产出 |
|------|------|---------|------|
| **Level 1** 会用AI工具 | 能熟练使用对话式AI完成日常任务 | 1-2周 | 会用提示词提问 |
| **Level 2** 搭建智能体 | 根据场景搭建个性化智能体 | 2-4周 | 1-3个实用智能体 |
| **Level 3** 编排工作流 | 让多个步骤自动化执行 | 1-2个月 | 工作流+完整流程 |
| **Level 4** 构建生态 | 全校推广，形成AI文化 | 持续 | 校本AI资源库 |

## 教师使用AI智能体的六个原则

### 1️⃣ 以问题为导向，不以技术为导向

<div class="comparison-table">

<div class="comparison-card bad">
<span class="case-label">❌ 错误思路</span>

"AI能做什么？我要用上。"

结果：为了用AI而用AI，增加了负担而不是减轻。

</div>

<div class="comparison-card good">
<span class="template-label">✅ 正确思路</span>

"我工作中有什么痛点？AI能不能帮我解决。"

结果：针对性解决问题，效率实实在在提升。

</div>

</div>

**正确的操作步骤**：

1. 列出你每周最耗时的 5 项工作
2. 分析哪些可以部分自动化
3. 针对性搭建智能体

### 2️⃣ 人机协作，不是人机替代

AI智能体的定位是**助手**，不是**替代品**。

| AI负责 | 教师负责 |
|--------|---------|
| 资料收集与整理 | 判断资料的相关性和准确性 |
| 初稿生成 | 修改、润色、定稿 |
| 数据计算与图表 | 数据解读和教学决策 |
| 批量重复性工作 | 创造性教学设计和情感关怀 |

### 3️⃣ 从小处开始，逐步扩展

不要一开始就搭建一个"万能智能体"。

**建议路径**：

```
第1周：搭建一个简单的教案生成智能体
  ↓
第2周：给它添加知识库（课标内容）
  ↓
第3周：连接出题工具，增加组卷功能
  ↓
第4周：优化人设和回复逻辑
  ↓
持续迭代...
```

### 4️⃣ 建立审核机制

AI生成的内容**必须经过人工审核**：

| 内容类型 | 审核重点 |
|----------|---------|
| 知识性内容 | 检查事实准确性 |
| 教学设计 | 确认符合学情 |
| 试题 | 验证答案正确性 |
| 对外发布内容 | 审核合规性 |

### 5️⃣ 保护数据安全

使用AI智能体时注意：

| ❌ 不要做 | ✅ 可以做 |
|----------|---------|
| 上传学生个人信息（姓名、身份证号） | 上传公开的教学资源 |
| 上传含敏感信息的内部文件 | 上传课程标准、教材 |
| 泄露教师账号密码 | 使用脱敏后的数据 |

### 6️⃣ 持续学习和分享

- 关注AI教育的最新发展
- 和同事分享好的智能体搭建经验
- 参与教师AI学习社群

## 常见误区

| 误区 | 正确理解 |
|------|---------|
| "AI什么都知道" | AI可能出错，需要审核 |
| "用了AI就不备课了" | AI辅助备课，不是替代备课 |
| "越复杂的智能体越好" | 够用就好，简单实用 |
| "学生不能用AI" | 要引导学生正确使用AI |
| "AI会取代教师" | AI是工具，教师的核心价值不可替代 |
