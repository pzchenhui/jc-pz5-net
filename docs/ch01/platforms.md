# 1.4 主流AI智能体搭建平台

## 平台选择的核心标准

选平台就像选学校——没有绝对最好的，只有最适合你的。

<div class="case-card">
<span class="case-label">📌 选错平台的教训</span>

### 李老师的教训

李老师一开始选了一个国外平台：
- 界面全是英文，操作困难
- 中文回复质量不稳定
- 分享给同事，同事打不开
- 用了两周就放弃了

**教训**：选平台要考虑语言、生态、分享便利性。

</div>

选之前想清楚这几个问题：

| 考量因素 | 说明 | 权重 |
|----------|------|------|
| **语言支持** | 中文支持好不好？ | ⭐⭐⭐⭐⭐ |
| **上手难度** | 零代码还是需要编程？ | ⭐⭐⭐⭐⭐ |
| **费用** | 免费额度够不够用？ | ⭐⭐⭐⭐ |
| **功能** | 知识库、插件、工作流是否完善？ | ⭐⭐⭐⭐ |
| **稳定性** | 服务是否稳定可靠？ | ⭐⭐⭐⭐ |
| **分享扩散** | 发布后其他人能不能方便使用？ | ⭐⭐⭐ |

## 主流平台详细介绍

<div class="comparison-card good" style="margin-bottom:16px">

### ⭐ 1. Coze（扣子）—— 零基础教师首选

**推荐指数**：⭐⭐⭐⭐⭐

**基本信息**：
- 开发方：字节跳动
- 官网：[www.coze.cn](https://www.coze.cn)
- 免费额度：充足（日常教学够用）

**核心功能**：

| 功能 | 支持情况 | 说明 |
|------|---------|------|
| 零代码搭建 | ✅ | 拖拽式操作 |
| 知识库 | ✅ | 支持PDF、Word、TXT、网页 |
| 插件市场 | ✅ | 50+插件（搜索、图片、文档等） |
| 工作流 | ✅ | 可视化编排 |
| 发布渠道 | ✅ | 微信公众号、飞书、Web、API |
| 团队协作 | ✅ | 团队空间共享智能体 |

**教师场景适配度**：

| 场景 | 适配度 | 说明 |
|------|--------|------|
| 教案生成 | ⭐⭐⭐⭐⭐ | 知识库+工作流完美适配 |
| 学科辅导 | ⭐⭐⭐⭐⭐ | 对话流+安全过滤 |
| 活动策划 | ⭐⭐⭐⭐⭐ | 搜索插件+文案生成 |
| 行政办公 | ⭐⭐⭐⭐ | 文档处理能力强 |
| 班级管理 | ⭐⭐⭐⭐ | 对话流适合批量操作 |

**适合人群**：全体教师，特别是零基础的教师

</div>

### 2. Dify —— 技术型学校推荐

**推荐指数**：⭐⭐⭐⭐

**基本信息**：
- 类型：开源（可自建）
- 官网：[dify.ai](https://dify.ai)
- 费用：开源免费（自建成本另算）

**核心功能**：

| 功能 | 支持情况 | 说明 |
|------|---------|------|
| 零代码搭建 | ✅ | 可视化界面 |
| 知识库 | ✅ | RAG检索增强生成 |
| 工作流 | ✅ | 功能强大 |
| API接口 | ✅ | 完善，方便集成 |
| 自建部署 | ✅ | 数据完全可控 |

**适合场景**：
- 学校级AI平台（可自建部署）
- 需要对接校内系统的场景
- 对数据安全要求高的场景

**注意**：自建需要技术人员维护，适合有IT团队的学校。

</div>

## 本教程的选择

> 📌 **本教程以 Coze（扣子）为主要演示平台。**

**为什么选 Coze？**

1. 对零基础教师最友好
2. 功能完整（知识库、插件、工作流全覆盖）
3. 免费额度够用
4. 中文教学场景支持好
5. 发布渠道丰富（微信公众号、飞书、Web）

但教程中的思路和方法论**适用于所有平台**。学完后你可以自由迁移到其他平台。
