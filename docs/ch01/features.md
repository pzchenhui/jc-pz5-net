# 1.2 AI智能体可以实现哪些功能？

## 先看两个真实场景

<div class="case-card">

<span class="case-label">📌 案例一：王老师的备课日常</span>

王老师是初中语文老师，教两个班的语文。

**以前备课需要：**
- ❌ 翻教材、查资料 → 1小时
- ❌ 写教案 → 2小时
- ❌ 做PPT → 1.5小时
- ❌ 出练习题 → 1小时
- **合计：约 5.5 小时**

**有了AI智能体后：**
- ✅ 告诉智能体课题 → 自动生成教案 → 15分钟
- ✅ 智能体基于教案生成PPT大纲 → 10分钟
- ✅ 智能体按知识点出题 → 15分钟
- **合计：约 40 分钟**

> ⏱️ 节省了 **4小时40分钟**，王老师用这些时间研究了两个学困生的个性化辅导方案。

</div>

<div class="case-card">

<span class="case-label">📌 案例二：李主任的行政工作</span>

李主任是学校办公室主任，每学期开学要准备：
- 新学期工作计划
- 各部门工作安排汇总
- 开学典礼方案
- 安全隐患排查表

**以前**：收集各部门材料→汇总→排版，**耗时一周**。

**现在**：通过AI智能体：
1. 提前设定好模板和流程
2. 各部门在线提交信息
3. 智能体自动汇总、格式统一
4. 生成完整文档

> ⏱️ **从一周缩短到半天。**

</div>

## AI智能体的五大核心能力

### 1️⃣ 内容生成

<div class="template-card">

<span class="template-label">📝 适用场景</span>

| 能做什么 | 教师场景 | 实操提示词示例 |
|----------|---------|---------------|
| 文本书写 | 教案、通知、总结、评语 | "帮我写一份三年级家长会发言稿" |
| 题目生成 | 选择题、填空题、整卷 | "出10道关于分数的选择题" |
| 方案设计 | 活动策划、教学方案 | "策划一场校园科技节活动方案" |
| 多语言翻译 | 英语教学设计 | "将这篇教案翻译成英文" |

</div>

### 2️⃣ 知识检索

| 能做什么 | 教师场景 | 实操提示词示例 |
|----------|---------|---------------|
| 政策查询 | 教育法规、课程标准 | "课标中对小学阅读有什么要求？" |
| 资料查找 | 学科知识、教学案例 | "找一些《草船借箭》的教学设计" |
| 数据分析 | 成绩统计、学情分析 | "分析这个班的成绩分布情况" |

### 3️⃣ 流程自动化

| 能做什么 | 教师场景 |
|----------|---------|
| 文档处理 | 批量生成学生评语、成绩报告 |
| 信息收集 | 自动汇总家长问卷、反馈表 |
| 定时任务 | 每天推送教学提醒、每周生成周报 |

### 4️⃣ 交互对话

| 能做什么 | 教师场景 |
|----------|---------|
| 学生答疑 | 7×24 回答学生学科问题 |
| 家长沟通 | 自动回复常见问题 |
| 心理辅导 | 初步识别学生情绪问题 |

### 5️⃣ 多工具协作

| 能做什么 | 教师场景 |
|----------|---------|
| 连接搜索引擎 | 查找最新教学资源 |
| 连接文档工具 | 自动生成 Word/PPT 文件 |
| 连接数据源 | 读取教务系统数据进行分析 |

## 按教师角色划分功能清单

### 👩‍🏫 学科教师

- 📝 教案生成与优化
- ✍️ 试题命制与组卷
- 📊 学情分析报告
- 🎨 课件大纲设计
- 📚 知识点梳理

### 👤 班主任

- 📋 学生成长档案管理
- 💬 家校沟通模板
- 📋 班级工作计划
- 🎯 主题班会方案
- ❤️ 学生心理关怀

### 📋 行政人员

- 📄 公文写作
- 📅 会议安排与纪要
- 📊 数据汇总报表
- 🔍 信息检索整理
- 📢 通知发布

## ⚠️ 能力边界——AI智能体不能做什么？

了解边界同样重要：

| ❌ 不能做 | 原因 |
|----------|------|
| 替代教师的情感关怀 | AI没有真实情感 |
| 承担教育决策的最终责任 | 决策权在人 |
| 100% 保证内容准确性 | 可能出现"幻觉"，需人工审核 |
| 处理涉及隐私的敏感数据 | 需遵守数据保护法规 |

> ⚠️ **重要提醒**：AI智能体是辅助工具，不是替代品。教师的判断力、专业性和教育情怀是无法被替代的。

## 📋 本章小结

| 核心能力 | 一句话总结 |
|----------|-----------|
| 内容生成 | 帮你写教案、出题、做方案 |
| 知识检索 | 帮你查课标、找资料、分析数据 |
| 流程自动化 | 帮你批量处理重复性工作 |
| 交互对话 | 帮你答疑、沟通、关怀 |
| 多工具协作 | 帮你串联多个工具完成复杂任务 |

## ✏️ 思考练习

2. **算时间**：估算如果用了AI智能体，你能节省多少时间？
3. **优先级**：从节省时间最多的开始，排序你最想用AI解决的工作。
