# 1.1 普通AI工具 vs AI智能体

## 你已经用过的AI工具

如果你用过 DeepSeek、文心一言、通义千问等对话工具，或者用过 AI 生成图片、翻译文字、写邮件摘要——恭喜你，你已经体验过 AI 工具了。

但这些，只是 AI 的"入门级"应用。

## 普通AI工具长什么样？

**普通AI工具**的工作方式很简单：

```
你提问 → AI回答 → 对话结束
```

就像你去商店买东西：告诉店员你要什么，店员递给你，交易完成。

### 📋 典型场景

| 场景 | 工具 | 操作 |
|------|------|------|
| 写一段通知 | DeepSeek | 输入"帮我写一份家长会通知"，复制结果 |
| 翻译一篇文章 | DeepL | 粘贴原文，获取译文 |
| 生成一张图片 | Midjourney | 输入描述词，下载图片 |

**特点**：每次操作是独立的，AI不会记住你的偏好，不会主动帮你做下一步。

## 🤖 AI智能体又是什么？

**AI智能体（AI Agent）**是一个有"脑子"的AI。它不仅仅是回答你的问题，而是能：

```
你给出目标 → 智能体自己规划步骤 → 调用各种工具 → 完成整套任务 → 交付结果
```

就像你雇了一个助手：告诉它目标，它会自己想办法完成，过程中还会问你确认关键决定。

### 核心区别对比

| 对比维度 | 普通AI工具 | AI智能体 |
|----------|-----------|---------|
| **工作模式** | 一问一答 | 自主规划+多步执行 |
| **记忆能力** | 每次对话独立 | 可记住你的偏好和上下文 |
| **工具调用** | 不能或有限 | 可连接外部工具和API |
| **个性化** | 通用回答 | 根据你的场景定制 |
| **主动性** | 被动回答 | 可主动提醒、推荐 |

## 💡 用一个比喻来理解

> 🎯 **普通AI工具** = 一本超级百科全书——你有问题就去查，查完就走。
>
> 🤖 **AI智能体** = 一个专属助理——你告诉它目标，它帮你规划、执行、跟进。

### 普通AI工具的使用方式

张老师打开 DeepSeek，输入：
> "帮我出一份五年级数学第三单元的测试题。"

它给了一份试卷，张老师需要自己：
1. 复制到 Word 文档
2. 调整格式和字体
3. 加上学校抬头和考试说明
4. 手写加上答题区域
5. 检查题目是否符合课标要求

**总耗时：约 45 分钟**

### AI智能体的使用方式

张老师告诉智能体：
> "帮我准备五年级数学第三单元的教学材料。"

智能体会自动完成：
1. 📋 **查课标**——从知识库调取课程标准中对应的知识点要求
2. 📝 **写教案**——根据知识点编写教学设计（含教学目标、重难点、教学过程）
3. ✍️ **出试卷**——按难度梯度出10道选择题、5道填空题、3道应用题，附答案
4. 📊 **做课件**——生成PPT大纲，标注每页内容和动画建议
5. 💾 **打包整理**——把所有材料整理成统一格式，加上学校抬头

**总耗时：约 15 分钟（含人工审核）**

> ⏱️ **节省了 30 分钟，效率提升约 3 倍！**

## 为什么要用智能体？

对教师来说，AI智能体的核心价值在于：

| 价值 | 说明 | 实际效果 |
|------|------|---------|
| **减少重复劳动** | 把常规性工作交给智能体 | 每周节省 5-10 小时 |
| **保持一致性** | 智能体按你的标准输出 | 教案格式统一、质量稳定 |
| **释放创造力** | 从繁琐事务中解脱 | 专注教学设计和学生关怀 |
| **可传承分享** | 好的智能体分享给同事 | 形成校本AI资源库 |

## 📋 本章小结

| 要点 | 说明 |
|------|------|
| 普通AI工具 | 回答单次问题，用完即走，每次独立 |
| AI智能体 | 有记忆、能规划、可调用工具、个性化定制 |
| 教师价值 | 减少重复劳动，保持一致性，释放创造力 |

3. **规划**：如果有一个专属AI助手，你最想让它帮你做什么？
