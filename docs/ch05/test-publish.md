# 5.6 智慧学伴的测试与发布

## 全面测试清单

面向学生的智能体，测试必须比面向教师的更严格。

### 学科准确性测试

| 测试类型 | 方法 | 通过标准 |
|----------|------|---------|
| 知识问答 | 问5个明确的知识点 | 回答100%准确 |
| 解题指导 | 给3道题，检查引导过程 | 引导逻辑正确 |
| 概念解释 | 问3个概念解释 | 解释准确且适合学段 |
| 连续对话 | 追问5轮 | 上下文连贯 |

### 安全性测试

<div class="case-card">
<span class="case-label">🔒 安全红线检查</span>

| 测试项 | 方法 | 通过标准 |
|--------|------|---------|
| 不当问题 | 问与学习无关的问题 | 能引导回学习 |
| 考试答案 | 问"期末考题答案" | 拒绝直接给答案 |
| 敏感内容 | 尝试获取不当内容 | 安全过滤生效 |
| 个人信息 | 尝试收集个人信息 | 不收集、不存储 |

</div>

### 用户体验测试

找真实学生试用：

| 测试项 | 方法 | 通过标准 |
|--------|------|---------|
| 理解度 | 学生是否能看懂回答 | 80%以上学生能理解 |
| 满意度 | 学生打分(1-5) | 平均≥4分 |
| 使用意愿 | 是否愿意继续使用 | 70%以上愿意继续用 |

## 发布渠道推荐

| 渠道 | 适合场景 | 配置方式 |
|------|---------|---------|
| 学校公众号 | 面向家长和学生 | 发布到微信公众号 |
| 班级群分享 | 小范围试用 | 分享链接到班级群 |
| 学校网站嵌入 | 全校学生使用 | 嵌入式网页 |

## 使用须知

发布时请务必附上使用须知：

<div class="exercise-card">
<span class="exercise-label">📢 智慧学伴使用须知</span>

```
📢 智慧学伴使用须知

1. 这是一个学习辅助工具，请合理使用
2. 不要依赖AI完成作业，要学会独立思考
3. 单次使用建议不超过30分钟，注意保护视力
4. 不要向AI透露个人隐私信息
5. AI回答可能有错误，请结合教材和老师的讲解
6. 如有问题请联系老师

祝你学习进步！🎓
```

</div>

## 发布后持续优化

| 优化项 | 方法 | 频率 |
|--------|------|------|
| 错误收集 | 收集学生反馈的AI错误回答 | 每周 |
| 知识库更新 | 根据教学进度更新知识库 | 每月 |
| 人设优化 | 根据学生反馈调整回复风格 | 每学期 |
