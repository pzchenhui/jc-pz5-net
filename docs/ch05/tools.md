# 5.7 巧用AI工具：与智慧学伴携手同行

## AI学习工具矩阵

智慧学伴搭建完成后，还可以配合其他AI工具，形成更完整的学习支持体系：

<div class="template-card">
<span class="template-label">📝 学习工具推荐</span>

| 工具类型 | 推荐工具 | 用途 | 适合学段 |

| 智慧学伴 | 自建智能体 | 日常答疑、练习 | 全学段 |

## 📋 建议学习流程

```
课前预习
  ↓ 用智慧学伴了解新知识
  ↓ 用思维导图整理知识框架
  
课堂学习
  ↓ 认真听讲，记笔记
  ↓ 标记不懂的地方
  
课后复习
  ↓ 用智慧学伴解答疑惑
  ↓ 用AI工具检查作业
  ↓ 用错题本整理错题
  
定期总结
  ↓ 用智慧学伴生成单元总结
  ↓ 更新思维导图
```

## ⚠️ 使用提醒

| 提醒 | 说明 |
|------|------|
| 独立思考 | AI是辅助工具，不是替代品 |
| 验证答案 | AI可能有错，重要问题请教老师 |
| 时间管理 | 单次使用不超过30分钟 |
| 保护视力 | 注意用眼卫生 |
