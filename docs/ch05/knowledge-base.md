# 5.4 智慧学伴的知识库

## 知识库的重要性

对于面向学生的智慧学伴，知识库决定了它**能回答什么水平的问题**以及**回答是否准确**。

<div class="case-card">
<span class="case-label">📌 对比案例</span>

### 有知识库 vs 无知识库

**无知识库的AI**回答学生"什么是光合作用？"：
> 光合作用是植物利用阳光将二氧化碳和水转化为有机物和氧气的过程。
> 
> （回答正确但太简单，没有结合教材内容）

**有知识库的AI**（上传了教材和课标）：
> 光合作用是绿色植物通过叶绿体，利用光能，把二氧化碳和水转化成储存着能量的有机物（主要是淀粉），并且释放出氧气的过程。
> 
> 📖 这是七年级上册第三单元的内容。根据课程标准，你需要：
> 1. 能说出光合作用的原料和产物
> 2. 能描述光合作用的基本过程
> 3. 理解光合作用对生物圈的意义
> 
> 要不要我来考考你？

</div>

## 📋 推荐上传的内容

| 内容类型 | 示例文件 | 说明 |
|----------|---------|------|
| 教材内容 | 电子教材PDF | 确保回答在教材范围内 |
| 课程标准 | 课标文档PDF | 确保目标不超纲 |
| 知识点梳理 | 知识导图/总结文档 | 帮助精准定位知识点 |
| 常见错题集 | 学生常犯错误汇总 | 有针对性地解答和提醒 |
| 优秀解题方法 | 典型例题及解析 | 提供标准解题思路 |

## 知识库组织建议

### 按学科分类

```
📁 智慧学伴知识库/
  ├── 📁 数学/
  │    ├── 七年级教材.pdf
  │    ├── 八年级教材.pdf
  │    ├── 课程标准.pdf
  │    └── 常见错题集.pdf
  ├── 📁 语文/
  │    ├── 课文原文.txt
  │    ├── 知识点梳理.pdf
  │    └── 作文范例.pdf
  └── 📁 英语/
       ├── 教材电子本.pdf
       └── 语法总结.pdf
```

### 按学段分类

如果要做跨学科的智慧学伴，建议按学段组织知识库，避免小学生问到高中内容。

## 📋 实操步骤

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">准备文件</span>

收集教材、课标、知识点梳理等文件，按学科分类整理。
</div>

<div class="step-card">
<span class="step-number">2</span>
<span class="step-title">上传到知识库</span>

1. 在智能体配置页面找到"知识库"
2. 点击"添加知识库"
3. 选择"上传文件"
4. 按学科分别上传
5. 等待文件处理完成
</div>

<div class="step-card">
<span class="step-number">3</span>
<span class="step-title">测试效果</span>

问一个与知识库内容相关的问题，检查：
- 回答是否引用了教材内容
- 是否在课标范围内
- 是否准确
</div>

## 知识库使用技巧

| 技巧 | 说明 |
|------|------|
| 精准检索 | 设置知识库检索方式为"精准"，避免答非所问 |
| 多知识库 | 不同学科使用不同知识库 |
| 定期更新 | 教材改版后及时更新知识库 |
| 控制数量 | 不要一次性上传过多文件，影响检索效果 |
