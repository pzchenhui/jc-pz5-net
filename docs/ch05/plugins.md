# 5.3 智慧学伴插件的选择与配置

## 智慧学伴需要的插件

相比备课助手，智慧学伴对插件的需求有所不同——面向学生使用，更注重**互动性和安全性**。

<div class="case-card">
<span class="case-label">📌 使用场景对比</span>

### 备课助手 vs 智慧学伴的插件差异

| 插件 | 备课助手 | 智慧学伴 | 原因 |
|------|---------|---------|------|
| 联网搜索 | 必须 | 可选 | 学生不需要实时搜索 |
| 计算器 | 可选 | 必须 | 数学计算验证答案 |
| 图片生成 | 可选 | 可选 | 辅助理解概念 |
| 代码执行 | 不需要 | 可选 | 编程类问题需要 |

</div>

## 推荐配置方案

### 小学学伴

| 插件 | 用途 | 配置建议 |
|------|------|---------|
| 计算器 | 验证数学题答案 | 仅用于验证，不展示计算过程给学生 |
| 图片生成 | 生成帮助理解的示意图 | 生成前先人工审核 |

### 初中学伴

| 插件 | 用途 | 配置建议 |
|------|------|---------|
| 联网搜索 | 查找时事素材、补充知识 | 限制搜索范围，过滤不当内容 |
| 计算器 | 数学/物理计算验证 | 展示计算步骤 |

### 高中学伴

| 插件 | 用途 | 配置建议 |
|------|------|---------|
| 联网搜索 | 补充时事素材、拓展知识 | 可放宽搜索范围 |
| 计算器 | 复杂计算验证 | 支持科学计算 |
| 代码执行 | 信息技术、编程题目 | 仅允许安全的代码运行 |

## 📋 实操步骤：添加插件

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">打开插件模块</span>

在智能体配置页面，找到 **"插件"** 模块，点击 **"添加插件"**
</div>

<div class="step-card">
<span class="step-number">2</span>
<span class="step-title">浏览插件市场</span>

在插件市场中搜索需要的插件：
- 搜索"计算器" → 选择适合的计算器插件
- 搜索"搜索" → 选择联网搜索插件
</div>

<div class="step-card">
<span class="step-number">3</span>
<span class="step-title">测试插件</span>

添加后在预览窗口测试：
- 计算器：输入"3.14 × 5²"，检查结果
- 搜索：输入"2024年诺贝尔文学奖"，检查搜索结果
</div>

## ⚠️ 面向学生的安全配置

| 安全措施 | 说明 |
|----------|------|
| 搜索结果过滤 | 过滤不适宜学生的内容 |
| 计算器限制 | 仅允许数学计算，不显示超纲内容 |
| 图片审核 | 生成的图片需要人工审核 |
| 对话监控 | 定期检查对话记录 |
