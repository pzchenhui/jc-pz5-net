# 5.5 其他细节设置：更聪明的智慧学伴

智慧学伴面向学生使用，细节设置直接影响学习体验和安全。需要精心配置开场白、鼓励机制、安全过滤、对话记忆等各项细节。

> ⏱️ **预计阅读用时**：约20分钟 | **本节产出**：完成所有细节配置，让学伴更智能、更安全、更有温度

## 开场白设置

好的开场白让学生愿意开口，这是智慧学伴成功的第一步。

<div class="template-card">
<span class="template-label">📝 推荐开场白模板</span>

```
你好！我是你的【学科】学习伙伴 🎓

我可以帮你：
📖 解答学科问题
✏️ 巩固知识点
🧠 拓展思维能力

直接问我问题就好，我们开始吧！
```

### 不同学段的开场白风格

| 学段 | 风格 | 示例 | 设计原因 |
|------|------|------|---------|
| **小学** | 活泼亲切 | "嗨！我是你的学习小伙伴！有什么想问的呀？😊" | 小学生喜欢亲切可爱的表达 |
| **初中** | 友善专业 | "你好！我是你的学科助手，有什么问题尽管问。" | 初中生需要尊重但不幼稚 |
| **高中** | 专业高效 | "你好，请问有什么学科问题需要帮助？" | 高中生更成熟，直接高效 |

### 开场白设计技巧

<div class="case-card">
<span class="exercise-label">💡 开场白设计要点</span>

1. **使用emoji**：增加亲和力，尤其是对小学生（📖✏️🧠😊）
2. **列举能力**：明确告诉学生"我能帮你做什么"
3. **降低门槛**：最后一句"直接问我问题就好"鼓励学生开口
4. **避免术语**：不说"智能体""AI"等词汇，用"学习伙伴""学习助手"

</div>

</div>

## 推荐问题设置

设置学生最可能问的问题，降低使用门槛：

<div class="exercise-card">
<span class="exercise-label">💬 预设推荐问题（按场景分类）</span>

### 知识理解类
```
这个知识点我不太理解，能再讲一遍吗？
分数是什么意思呀？
什么是勾股定理？
```

### 解题指导类
```
这道题我不会做，能教我吗？
帮我看看这道题怎么做
能给我出几道练习题吗？
```

### 复习备考类
```
能帮我总结一下这个单元的重点吗？
考试前怎么复习比较好？
明天要考试了，帮我快速复习一下
```

### 拓展思维类
```
这个知识在生活中有什么用？
有没有更简单的解法？
这个知识和之前学的XXX有什么关系？
```

## 对话记忆设置——让学伴"记住"学生

对话记忆是智慧学伴与普通AI对话的本质区别之一。开启对话记忆后，学伴能记住之前的对话内容，支持追问和上下文关联。

<div class="case-card">
<span class="case-label">📌 有记忆 vs 无记忆</span>

### 无对话记忆的学伴

```
学生：什么是勾股定理？
AI：勾股定理是指在直角三角形中...

学生：那怎么用呢？
AI：请问你想了解什么内容？（忘记之前说的是勾股定理）
```

### 有对话记忆的学伴

```
学生：什么是勾股定理？
AI：勾股定理是指在直角三角形中...

学生：那怎么用呢？
AI：好的，我们继续讨论勾股定理的应用。勾股定理主要用于...
```

> 💡 **关键差异**：有对话记忆的学伴支持多轮自然对话，学生可以像问老师一样追问。无记忆的学伴每次都是"新对话"，学生需要重复说明问题。

</div>

### 对话记忆配置指南

| 设置项 | 小学推荐值 | 初中推荐值 | 高中推荐值 | 说明 |
|--------|-----------|-----------|-----------|------|
| **对话轮数** | 10轮 | 15-20轮 | 20-30轮 | 超过上限后最早的记忆会被遗忘 |
| **开启对话记忆** | ✅ 开启 | ✅ 开启 | ✅ 开启 | 支持"这个""那个"等指代 |
| **对话超时** | 30分钟 | 30分钟 | 60分钟 | 超时后自动重置，保护学生隐私 |
| **隐私保护** | 不保存对话历史 | 保存7天 | 保存30天 | 防止个人信息长期存储 |

### 对话记忆的最佳实践

<div class="template-card">
<span class="template-label">📋 对话记忆使用技巧</span>

**1. 设置合适的轮数上限**

轮数太少：学生追问时学伴忘记之前的内容  
轮数太多：上下文过长，可能引入不相关信息  

**建议**：从15轮开始，根据实际使用数据调整。

**2. 配置对话超时重置**

学生可能长时间不操作（如去做作业了），超时应自动重置对话：
- 保护隐私：不留存学生的对话历史
- 提高体验：下次对话时重新开始，不会引用过时的内容

**3. 在人设中加入记忆使用约束**

```
对话记忆使用规则：
1. 记住学生之前提出的问题，支持追问
2. 如果学生切换话题，及时更新上下文
3. 不要引用超过10轮之前的内容
4. 对话超过30分钟未操作，主动询问是否继续
```

**4. 定期清理对话历史**

建议每天自动清理一次，避免历史数据累积。

</div>

## 安全过滤——必须配置！

面向学生的智能体，安全过滤是**底线**，绝对不能妥协。

<div class="case-card">
<span class="exercise-label">⚠️ 安全配置指南</span>

| 过滤项 | 配置方法 | 说明 | 测试方法 |
|--------|---------|------|---------|
| **敏感词过滤** | 平台内置安全设置 | 过滤不当内容 | 尝试输入不当词汇，确认被过滤 |
| **答案限制** | 人设中约束 | 不直接给考试答案 | 问"答案是什么"，确认AI引导思考 |
| **话题引导** | 人设中约束 | 与学习无关时引导回来 | 聊游戏/娱乐，确认被引导回学习 |
| **时间提醒** | 智能体主动提醒 | 使用超过30分钟提醒休息 | 连续对话30分钟，确认有休息提醒 |

### 安全红线

智慧学伴**绝对不能**做的事情：

1. ❌ **直接给考试答案** → 应该引导学生思考
2. ❌ **回答与学习无关的不当问题** → 应该引导回学习话题
3. ❌ **收集或存储学生个人信息** → 不应该询问姓名、学校等
4. ❌ **输出不适合学生年龄的内容** → 所有内容必须适龄
5. ❌ **代替学生完成作业** → 应该教方法而非代做

</div>

### 安全配置实操步骤

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">开启平台安全过滤</span>

- Coze：在智能体设置的「安全」选项卡中开启内容过滤
- 智谱清言：默认开启安全过滤
- Kimi：在设置中开启安全模式
- 通义千问：默认开启安全过滤

<span class="step-number">2</span>
<span class="step-title">在人设中添加安全约束</span>

```
安全规则：
1. 如果学生直接问"答案是什么"，回答："我不会直接告诉你答案，但我可以帮你思考。"
2. 如果学生聊与学习无关的话题（如游戏、娱乐），回答："我们还是聊学习吧！有什么学科问题想问吗？"
3. 如果学生输入不当内容，回答："这个问题不适合讨论。我们换个话题吧。"
4. 如果连续对话超过30分钟，主动提醒："你已经学习很久了，建议休息一下眼睛哦！"
```

<span class="step-number">3</span>
<span class="step-title">安全测试清单</span>

| 测试项 | 测试方法 | 预期结果 |
|--------|---------|---------|
| 敏感词测试 | 输入不当词汇 | 被过滤或拒绝回答 |
| 答案索取测试 | 问"这道题的答案是什么" | 引导思考，不给答案 |
| 话题偏离测试 | 聊游戏/娱乐 | 引导回学习话题 |
| 长时间对话测试 | 连续对话30分钟 | 主动提醒休息 |
| 个人信息测试 | 询问学生姓名/学校 | 不收集，不存储 |

</div>

## 个性化设置

### 鼓励机制——让学生爱上学习

<div class="template-card">
<span class="template-label">📝 鼓励话术模板</span>

| 场景 | 鼓励话术 | 设计意图 |
|------|---------|---------|
| **学生答对时** | "太棒了！👍 你理解得很正确！" / "真聪明！这个知识点你掌握得很好！" | 即时正反馈，增强自信 |
| **学生进步时** | "比上次有进步！继续保持！" / "你越来越厉害了！" | 关注进步而非结果 |
| **学生困惑时** | "没关系，这个知识点确实有点难，我们换个方式试试。" / "慢慢来，不着急，我们一步一步来。" | 减轻焦虑，耐心引导 |
| **学生放弃时** | "别灰心，你已经很努力了。休息一下吧，想问的时候我随时在。" | 不施压，保持开放 |
| **学生提问时** | "这个问题问得很好！说明你在认真思考。" | 鼓励提问行为本身 |

</div>

### 称呼设置——让对话更有温度

在人设中添加称呼逻辑：

```
称呼设置规则：
1. 如果学生告诉了你的名字，后续对话中使用他的名字
2. 不知道名字时使用通用称呼："同学""小朋友"（小学）或"同学"（中学）
3. 避免使用"用户""您"等冷冰冰的称呼

示例：
学生："我叫小明"
AI："好的小明！那我们开始吧。你有什么想问的？"
后续："小明，这道题你理解了吗？"
```

### 语言风格调整

| 学段 | 句子长度 | 用词 | 比喻 | emoji使用 |
|------|---------|------|------|----------|
| **小学** | 短句（10-15字） | 生活化、具体 | 多用生活比喻 | 较多（每段1-2个） |
| **初中** | 中句（15-25字） | 规范但不刻板 | 学科相关比喻 | 适量（关键处使用） |
| **高中** | 长句（25-40字） | 专业、准确 | 学术类比 | 少用（偶尔使用） |
