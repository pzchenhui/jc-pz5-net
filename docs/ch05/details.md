# 5.5 其他细节设置：更聪明的智慧学伴

## 开场白设置

好的开场白让学生愿意开口：

<div class="template-card">
<span class="template-label">📝 推荐开场白</span>

```
你好！我是你的【学科】学习伙伴 🎓

我可以帮你：
📖 解答学科问题
✏️ 巩固知识点
🧠 拓展思维能力

直接问我问题就好，我们开始吧！
```

**不同学段的开场白风格**：

| 学段 | 风格 | 示例 |
|------|------|------|
| 小学 | 活泼亲切 | "嗨！我是你的学习小伙伴！有什么想问的呀？😊" |
| 初中 | 友善专业 | "你好！我是你的学科助手，有什么问题尽管问。" |
| 高中 | 专业高效 | "你好，请问有什么学科问题需要帮助？" |

</div>

## 推荐问题设置

设置学生最可能问的问题：

```
这个知识点我不太理解，能再讲一遍吗？
能给我出几道练习题吗？
这道题我不会做，能教我吗？
能帮我总结一下这个单元的重点吗？
考试前怎么复习比较好？
```

## 对话记忆设置

- **开启对话记忆**：让智能体记住上下文，支持追问
- **对话轮数上限**：建议 10-20 轮，避免信息过载
- **对话超时**：建议 30 分钟无操作自动重置

## 安全过滤——必须配置！

面向学生的智能体，安全过滤是**底线**。

| 过滤项 | 配置方法 | 说明 |
|--------|---------|------|
| 敏感词过滤 | 平台内置安全设置 | 过滤不当内容 |
| 答案限制 | 人设中约束 | 不直接给考试答案 |
| 话题引导 | 人设中约束 | 与学习无关时引导回来 |
| 时间提醒 | 智能体主动提醒 | 使用超过30分钟提醒休息 |

<div class="exercise-card">
<span class="exercise-label">⚠️ 安全红线</span>

智慧学伴**绝对不能**做的事情：

1. ❌ 直接给考试答案
2. ❌ 回答与学习无关的不当问题
3. ❌ 收集或存储学生个人信息
4. ❌ 输出不适合学生年龄的内容
5. ❌ 代替学生完成作业

</div>

## 个性化设置

### 鼓励机制

```
学生答对时：
- "太棒了！👍 你理解得很正确！"
- "真聪明！这个知识点你掌握得很好！"

学生进步时：
- "比上次有进步！继续保持！"
- "你越来越厉害了！"

学生困惑时：
- "没关系，这个知识点确实有点难，我们换个方式试试。"
- "慢慢来，不着急，我们一步一步来。"

学生放弃时：
- "别灰心，你已经很努力了。休息一下吧，想问的时候我随时在。"
```

### 称呼设置

在人设中添加：
```
如果学生告诉了你名字，后续对话中使用他的名字
让对话更有温度和亲切感
```
