# 4.4 考试测评助手：快速生成习题和试卷

## 命题的痛点——为什么教师害怕出题？

出题是教师最耗时、最费脑的工作之一。每次考试都要经历"找题→组卷→验算→排版→答案"的完整流程，而且稍有疏忽就会出答案错误、超纲、难度不均等问题。

> ⏱️ **预计阅读用时**：约20分钟 | **本节产出**：搭建考试测评智能体，能按知识点/难度/题型生成试题和完整试卷

<div class="case-card">
<span class="case-label">📌 真实案例</span>

### 王老师的命题经历

王老师教初一数学，每次月考都要命题。

**以前命题（2024年）**：
- 翻教材、找题目 → 2小时
- 组合调整、控制难度 → 1.5小时
- 做答案、写评分标准 → 1小时
- 校对、排版 → 1小时
- **合计：约 5.5 小时**

**用考试测评智能体后（2025年）**：
- 输入考试范围和要求 → 2分钟
- AI生成试卷初稿 → 3分钟
- 人工审核修改、验算答案 → 30分钟
- **合计：约 35 分钟**

> ⏱️ **节省了 5 小时，效率提升约 9.4 倍！**

王老师说："最大的变化不是省了多少时间，而是我不再'怕'出题了。以前每次出卷都要熬半天，现在输入要求就能拿到一份质量不错的初稿，我只需要验算答案和微调难度就行。"

</div>

### 张老师的单元测验日常

张老师是小学英语教师，每周需要出一份单元小测（10题，15分钟）。

**AI辅助出题流程**：
1. 输入："出10道关于'一般过去时'的小测题，选择题6道+填空题4道，难度中等"
2. AI在30秒内生成题目+答案+解析
3. 张老师逐题验算（约5分钟）
4. 调整2道题的难度，补充1道情境题
5. 导出为Word文档，排版打印

**效果**：
- 出题时间从1.5小时→8分钟
- 题目质量稳定，答案正确率100%（经人工验算）
- 每周小测不再成为负担

## 📝 完整人设模板

<div class="template-card">
<span class="template-label">📝 命题专家完整人设模板</span>

```markdown
你是一位有15年命题经验的教研员，精通各学科考试命题
技术和布鲁姆认知目标分类（记忆→理解→应用→分析→评价→创造）。

## 你的任务
根据用户要求，生成高质量的习题和完整试卷。

## 命题原则
1. 依据课程标准和教材内容，绝对不超纲
2. 覆盖不同认知层次（至少包含记忆、理解、应用三个层次）
3. 难度梯度合理：基础题60%、中等题25%、挑战题15%
4. 情境真实，贴近学生生活实际
5. 答案100%准确，解析清晰明了
6. 每题标注考查的知识点和认知层次

## 出题格式要求

### 选择题
- 题干清晰，无歧义
- 4个选项（A/B/C/D），干扰项合理
- 标注正确答案
- 附简要解析

### 填空题
- 题目明确，答案唯一
- 标注答案和考查知识点

### 简答题/应用题
- 情境真实，有思维含量
- 附参考答案及评分要点
- 标注考查能力

## 试卷格式
XX学校 XX学年 第X学期 XX学科 X年级测试卷
满分：XX分  考试时间：XX分钟

一、选择题（每题X分，共X分）
二、填空题（每题X分，共X分）
三、简答题/应用题（每题X分，共X分）

注意事项：
- 试卷总分必须等于标明的满分
- 各题型分值之和必须等于该大题总分
- 难度标注：每道题注明"基础/中等/挑战"
```

</div>

## 📋 常用出题指令

掌握以下指令，可以覆盖日常出题的绝大部分场景：

<div class="template-card">
<span class="template-label">📝 出题指令速查表</span>

| 指令类型 | 提示词示例 | 适用场景 |
|----------|-----------|---------|
| **按知识点** | "出5道关于'分数加减法'的选择题，难度中等" | 课后练习、小测 |
| **按难度** | "出3道一元二次方程的挑战题，适合期末考试" | 培优、竞赛 |
| **按题型** | "出一份含选择题和填空题的单元测试，20题" | 单元测验 |
| **完整试卷** | "出一份七年级数学期末模拟试卷，满分120分，120分钟" | 月考、期末 |
| **变式训练** | "把这道题改成3个变式，难度递增" | 举一反三 |
| **双向细目表** | "请根据以下知识点生成双向细目表：..." | 命题规划 |
| **答案+解析** | "为以下题目生成答案和详细解析：..." | 补充答案 |
| **难度调整** | "把这套试卷的难度从'中等'调到'较难'" | 分层教学 |

</div>

### 出题指令的编写技巧

一条好的出题指令应包含以下要素：

```
【学科】+【年级/学段】+【知识点/范围】+【题型】+【数量】+【难度】

示例：
"七年级数学，关于'一元一次方程'的选择题8道，难度中等，每题附答案和解析"
```

## 命题质量审核表——AI出题后必做

**重要提醒**：AI生成的试卷**必须**经过人工审核才能使用。AI可能出错（尤其是数学计算），而且无法完全理解你的教学意图。

<div class="exercise-card">
<span class="exercise-label">✏️ AI出题后必做审核清单</span>

| 审核项 | 检查方法 | 通过标准 | 实际结果 | 通过 |
|--------|---------|---------|---------|------|
| **答案正确性** | 逐题验算 | 100%正确 | | □ |
| **不超纲** | 对照课标和教材 | 所有题目在范围内 | | □ |
| **难度合理** | 预估得分率 | 整体60-80% | | □ |
| **覆盖面** | 检查知识点分布 | 核心知识点全覆盖 | | □ |
| **表述清晰** | 逐题阅读题干 | 无歧义、无错字 | | □ |
| **分值合理** | 检查总分 | 等于试卷标明的满分 | | □ |
| **格式规范** | 检查排版 | 题号连续、格式统一 | | □ |
| **无重复题** | 对比往年试卷 | 重复率<20% | | □ |

**合格线**：以上8项全部通过，试卷可用于正式考试。

### 审核重点：数学题必须逐题验算

<div class="case-card">
<span class="case-label">⚠️ 特别注意</span>

**AI在数学计算中可能出错**，以下情况需要特别小心：

- **计算类题目**：每道题的答案都要手动验算
- **多步骤题目**：检查每一步的推导是否正确
- **单位换算题目**：检查单位是否统一、换算是否正确
- **实际应用题**：检查情境是否合理、数据是否真实

**建议**：数学试卷出题后，安排另一位数学教师交叉验算。

## 进阶：组卷工作流

如果你需要经常出题，可以搭建一个完整的组卷工作流，实现从"输入要求"到"输出试卷"的自动化：

<div class="template-card">
<span class="template-label">📋 自动组卷工作流</span>

```
输入：学科、年级、考试范围、满分、时间、难度分布
  ↓
节点1：生成双向细目表
  → 知识点 × 认知层次 × 难度
  → 确定每种题型的数量和分值
  ↓
节点2：按知识点生成题目
  → 调用大模型节点，逐知识点出题
  → 每题附答案和解析
  ↓
节点3：难度检查
  → 统计基础/中等/挑战题比例
  → 如果比例不符合要求，重新生成
  ↓
节点4：自动组卷
  → 按题型排序（选择→填空→解答）
  → 分配题号和分值
  → 检查总分是否等于满分
  ↓
节点5：生成试卷+答案
  → 输出完整试卷（含题目）
  → 输出答案和评分标准
  ↓
结束：输出完整试卷文档
```

**工作流优势**：
- 一次搭建，反复使用
- 确保每次出题都符合难度分布要求
- 自动生成双向细目表，便于命题分析

</div>

## ⚠️ 注意事项

| 注意项 | 说明 | 建议 |
|--------|------|------|
| **必须验算** | AI可能出错，尤其数学计算 | 每道题都要人工验算答案 |
| **避免超纲** | AI不知道你的教学进度 | 明确指定考试范围和知识点 |
| **保护原创** | 重要考试建议人工命题为主 | 期末、中考模拟等以人工为主，AI辅助 |
| **定期更新** | 避免与往年试卷重复率过高 | 每年更新题库，替换旧题 |
| **难度控制** | AI可能生成过难或过易的题 | 生成后评估预估得分率 |
| **公平性** | 题目不能有性别、地域歧视 | 检查题目表述是否公平 |

## 常见问题与解决

<div class="case-card">
<span class="exercise-label">❓ 常见问题</span>

### 问题1：AI出的题答案有误

**症状**：数学计算题的答案不正确。

**解决**：
1. **每道题必须人工验算**（这是不可省略的步骤）
2. 在提示词中加入"请逐步展示计算过程"
3. 对于复杂计算题，要求AI"用两种方法验证答案"

### 问题2：题目超纲

**症状**：AI生成了超出教学进度的题目。

**解决**：
1. 在提示词中明确指定知识点范围
2. 上传课程标准到知识库
3. 在提示词中加入"不要超出七年级上册第一章的范围"

### 问题3：题目难度不均

**症状**：一套试卷中有些题太简单，有些题太难。

**解决**：
1. 在提示词中指定难度分布（基础60%、中等25%、挑战15%）
2. 每道题标注难度等级
3. 生成后人工评估预估得分率

### 问题4：题目重复率过高

**症状**：AI出的题与往年试卷或教材习题高度相似。

**解决**：
1. 在提示词中加入"请原创题目，不要与教材习题雷同"
2. 上传往年试卷到知识库，要求AI避免重复
3. 使用"变式训练"指令改造旧题
