# 4.4 考试测评助手：快速生成习题和试卷

## 命题的痛点

<div class="case-card">
<span class="case-label">📌 真实案例</span>

### 王老师的命题经历

王老师教初一数学，每次月考都要命题。

**以前命题**：
- 翻教材、找题目 → 2小时
- 组合调整、控制难度 → 1.5小时
- 做答案、写评分标准 → 1小时
- 校对、排版 → 1小时
- **合计：约 5.5 小时**

**用考试测评智能体后**：
- 输入考试范围和要求 → 2分钟
- AI生成试卷初稿 → 3分钟
- 人工审核修改、验算答案 → 30分钟
- **合计：约 35 分钟**

> ⏱️ **节省了 5 小时，效率提升约 9.4 倍！**

## 📝 完整人设模板

<div class="template-card">
<span class="template-label">📝 命题专家人设模板</span>

```markdown
你是一位有丰富命题经验的教研员，精通各学科考试命题
技术和布鲁姆认知目标分类。

## 你的任务
根据用户要求，生成高质量的习题和试卷。

## 命题原则
1. 依据课程标准和教材内容，不超纲
2. 覆盖不同认知层次（记忆、理解、应用、分析、评价、创造）
3. 难度梯度合理：基础题60%、中等题25%、挑战题15%
4. 情境真实，贴近学生生活
5. 答案准确，解析清晰

## 输出格式

### 选择题
每题包含：题干、4个选项（A/B/C/D）、正确答案、解析

### 填空题
每题包含：题目、答案、考查知识点

### 简答题/应用题
每题包含：题目、参考答案及评分要点、考查能力

## 试卷格式
XX学校 XX学年 第X学期 XX学科 X年级测试卷
满分：XX分  时间：XX分钟
一、选择题（每题X分，共X分）
二、填空题（每题X分，共X分）
三、简答题/应用题（每题X分，共X分）
```
</div>

## 📋 常用出题指令

| 指令类型 | 提示词示例 |
|----------|-----------|
| 按知识点 | "出5道关于'分数加减法'的选择题" |
| 按难度 | "出3道一元二次方程的挑战题" |
| 按题型 | "出一份含选择题和填空题的单元测试" |
| 完整试卷 | "出一份七年级数学期末模拟试卷，满分100分" |
| 变式训练 | "把这道题改成3个变式，难度递增" |
| 双向细目表 | "请根据知识点生成双向细目表" |

## 命题质量审核表

<div class="exercise-card">
<span class="exercise-label">✏️ AI出题后必做审核</span>

AI生成的试卷**必须**经过人工审核才能使用：

| 审核项 | 检查方法 | 通过标准 |
|--------|---------|---------|
| 答案正确性 | 逐题验算 | 100%正确 |
| 不超纲 | 对照课标和教材 | 所有题目在范围内 |
| 难度合理 | 预估得分率 | 整体60-80% |
| 覆盖面 | 检查知识点分布 | 核心知识点全覆盖 |
| 表述清晰 | 逐题阅读题干 | 无歧义 |
| 分值合理 | 检查总分 | 等于试卷标明的满分 |

</div>

## 进阶：组卷工作流

```
输入：学科、年级、考试范围、满分、时间
  ↓
生成双向细目表（知识点×认知层次×难度）
  ↓
按细目表逐知识点生成题目
  ↓
检查难度分布和知识覆盖度
  ↓
自动组卷（选择题→填空题→解答题）
  ↓
生成答案和评分标准
  ↓
输出完整试卷
```

## ⚠️ 注意事项

| 注意 | 说明 |
|------|------|
| 必须验算 | AI可能出错，每道题都要人工验算 |
| 避免超纲 | 确认题目在课标和教材范围内 |
| 保护原创 | 重要考试（期末、中考模拟）建议人工命题为主 |
| 定期更新 | 避免与往年试卷题目重复率过高 |
