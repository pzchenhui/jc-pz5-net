# 4.5 巧用AI工具：快速生成课件

## 课件制作的AI工具生态

制作课件不只是文字工作，还涉及排版、配图、动画。AI工具可以帮你：

| 环节 | AI工具 | 用途 |
|------|--------|------|
| 内容生成 | 智能体 | 生成课件文字内容、大纲 |
| PPT生成 | Gamma / 讯飞智文 / WPS AI | 一键生成PPT文件 |
| 配图生成 | 通义万相 / DALL·E | 生成课件插图 |
| 排版优化 | WPS AI | 自动排版、美化 |

赵老师教初三化学，每周要做2-3个新课件。

## 实操步骤详解

### 第1步：用智能体生成课件大纲

<div class="template-card">
<span class="template-label">📝 课件大纲生成提示词</span>

```
请根据以下教案，提取课件大纲：

教案内容：（粘贴你的教案）

要求：
1. 每页PPT一个知识点或活动
2. 每页不超过6行文字
3. 标注建议的配图方向（如"此处配一张分子运动示意图"）
4. 标注动画建议（如"逐步出现""点击显示答案"）
5. 总页数控制在15-20页
6. 标注重点页面（需要花时间讲解的）
```

**输出示例**：
```
PPT大纲：七年级数学"一元一次方程"

1. 【封面】一元一次方程 | 配图：数学符号装饰背景
2. 【导入】生活中的方程 → 天平、购物找零
3. 【概念】什么是方程？ → 配图：天平示意图（重点）
4. 【概念】什么是一元一次方程？ → 对比：一元一次 vs 二元一次
5. 【判断】这些是不是方程？ → 动画：逐题出现
6. 【方法】解方程的基本步骤 → 配图：流程图（重点）
7. 【例题】例1：2x+3=7 → 动画：逐步展示解题过程
8. 【练习】试一试 → 点击显示答案
...
```

### 第2步：使用PPT生成工具

<div class="template-card">
<span class="template-label">🔧 推荐工具对比</span>

| 工具 | 优点 | 缺点 | 适合 |
|------|------|------|------|
| **Gamma** | 操作简单，AI生成质量高 | 中文模板较少 | 快速出初稿 |
| **讯飞智文** | 中文支持好，模板多 | 免费版有水印 | 中文课件 |
| **WPS AI** | 国内办公生态，可直接编辑 | 需要会员 | 日常办公 |
| **MindShow** | Markdown转PPT | 排版较简单 | 简洁风格 |

</div>

#### Gamma 实操

1. 打开 [gamma.app](https://gamma.app)
2. 点击 **"Create New"** → **"Generate"**
3. 粘贴课件大纲
4. 选择模板风格（教育类推荐"Professional"或"Creative"）
5. 等待生成（约30秒）
6. 下载为PPT或PDF

#### 讯飞智文 实操

1. 打开讯飞智文
2. 选择 **"AI生成PPT"**
3. 输入主题或粘贴大纲
4. 选择模板风格
5. 生成并下载

### 第3步：人工调整

AI生成的PPT是初稿，**必须**人工调整：

<div class="exercise-card">
<span class="exercise-label">✏️ 课件质量检查清单</span>

- [ ] 内容准确性（AI可能出错）
- [ ] 字体大小（正文不小于24号）
- [ ] 配色是否适合投影（对比度足够）
- [ ] 配图是否准确反映知识点
- [ ] 动画是否合理（不要太多太花）
- [ ] 每页信息量是否适中
- [ ] 有无错别字
- [ ] 页码是否正确

</div>

## 课件设计原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 少即是多 | 每页不超过6行文字 | 用关键词+口头讲解 |
| 图胜于文 | 能用图表示的尽量用图 | 分子运动用动画展示 |
| 一致性 | 字体、颜色、风格统一 | 全文2种字体、3种颜色 |
| 可读性 | 背景与文字对比度足够 | 白底黑字或深底白字 |
