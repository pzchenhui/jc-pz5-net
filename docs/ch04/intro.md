# 4.1 日常备课：怎样融入智能体？

## 备课是教师最核心的工作

备课占据了教师 **30-50%** 的工作时间。也是 AI 智能体最能发挥价值的场景。

<div class="case-card">
<span class="case-label">📌 数据说话</span>

### 教师备课时间调查

一项针对全国 2000 名中小学教师的调查显示：

| 备课环节 | 平均耗时 | 占比 |
|----------|---------|------|
| 研究教材和课标 | 45 分钟 | 20% |
| 编写教案 | 60 分钟 | 27% |
| 制作课件 | 40 分钟 | 18% |
| 准备练习题 | 35 分钟 | 16% |
| 其他（教具、板书等） | 20 分钟 | 9% |
| **合计** | **约 3.3 小时/课时** | **100%** |

> 一位教 2 个班的教师，每周备课时间约 **15-20 小时**。

</div>

## AI智能体在备课中的定位

| 备课环节 | AI能做什么 | 教师做什么 |
|----------|-----------|-----------|
| 研究教材 | 整理知识点、查课标 | 判断教学重点 |
| 编写教案 | 生成教案初稿 | 根据学情调整 |
| 制作课件 | 生成PPT大纲和初稿 | 调整视觉和动画 |
| 准备练习 | 按难度出题 | 审核答案、筛选 |

> 💡 **核心原则**：AI 生成初稿，教师审核定稿。

## 📋 智能体融入备课的三种方式

### 方式一：辅助型——从最难的部分开始

**适合**：刚开始接触AI的教师

```
你：帮我生成这个课题的教学目标
AI：根据课标，建议以下目标...
你：很好，我在此基础上调整
```

> 让AI帮你完成备课中**最耗时的部分**。

### 方式二：协同型——全程参与

**适合**：有一定AI使用经验的教师

```
你：我要备三年级语文《秋天的雨》这节课
AI：好的，我来帮你完成全流程：
    1. 先查课标 → 找到了相关要求
    2. 生成教案初稿 → 完成
    3. 生成练习题 → 完成
    4. 生成课件大纲 → 完成
    你看一下，哪里需要调整？
```

### 方式三：自动化型——一键备课

**适合**：已搭建工作流的教师

```
你：一键备七年级数学"一元一次方程"的课
AI：（自动执行工作流）
    教案 ✓ 练习题 ✓ 课件大纲 ✓
    完整教学包已生成，请审核
```

## 📋 备课工作流全景

```
确定课题
    ↓
AI查课标 → 确认知识点和目标要求
    ↓
AI生成教案 → 教师审核调整
    ↓
AI出练习题 → 教师验证答案
    ↓
AI做课件 → 教师调整美化
    ↓
完成备课 ✓
```
