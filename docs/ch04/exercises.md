# 📝 第4章练习：备课助手

## 练习目标

搭建完整的备课助手智能体，能生成教案、教学设计和试题。

---

## 🛠️ 实操任务：搭建备课助手

### 任务说明

使用第4章学到的知识，搭建一个完整的备课助手，至少包含教案生成和试题生成两个功能。

### 步骤检查清单

| 步骤 | 完成标准 | 打勾 |
|------|---------|------|
| 1. 编写备课人设 | 使用第4.2节的模板 | □ |
| 2. 配置课标知识库 | 上传至少1个学科的课标 | □ |
| 3. 设置教案生成功能 | 能生成完整格式教案 | □ |
| 4. 设置试题生成功能 | 能按题型和难度出题 | □ |
| 5. 测试5个不同课题 | 验证回复质量 | □ |
| 6. 发布并分享 | 分享给至少1位同事 | □ |
