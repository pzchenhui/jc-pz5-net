# 2.1 测一测：你急需哪类智能体？

## 先别急着动手，想清楚你要什么

很多老师一开始就冲进平台开始搭建，结果做出来的智能体不知道给谁用、解决什么问题。

花 5 分钟做完这个测试，找到你最需要的智能体类型。

## 自我诊断测试

### 第一步：你每周花最多时间的事是什么？

请在最符合的选项上打勾：

- [ ] A. 写教案、做教学设计
- [ ] B. 出题、改卷子、写评语
- [ ] C. 准备课件和教学材料
- [ ] D. 策划班级活动、写活动方案
- [ ] E. 处理行政事务、写报告
- [ ] F. 与家长沟通
- [ ] G. 回答学生问题
- [ ] H. 整理资料、汇总数据

### 第二步：你希望AI帮你解决什么问题？

请选择最迫切的 2-3 项：

- [ ] 1. 减少重复性文字工作
- [ ] 2. 提高备课效率
- [ ] 3. 保证教学内容的规范性
- [ ] 4. 快速响应学生提问
- [ ] 5. 减轻行政负担
- [ ] 6. 提升课堂互动性
- [ ] 7. 个性化辅导学生
- [ ] 8. 管理班级事务

### 第三步：对照诊断表

根据你前两步的选择，找到推荐的智能体类型：

| 你的情况 | 推荐智能体 | 对应章节 |
|----------|-----------|---------|
| 选 A + 选 1,2,3 | 📝 **备课助手** | 第4章 |
| 选 G + 选 4,7 | 📚 **智慧学伴** | 第5章 |
| 选 D + 选 1,6 | 🎉 **活动助手** | 第6章 |
| 选 E + 选 1,5 | 📋 **行政助手** | 第7章 |
| 选 F + 选 4,8 | 👤 **班主任助手** | 第8章 |
| 选 B + 选 1,2,3 | 📝 **备课助手**（测评模块） | 第4.4节 |
| 选 C + 选 1,2 | 📝 **备课助手**（课件模块） | 第4.5节 |
| 选 H + 选 1,5 | 📋 **行政助手** | 第7章 |

## 三种常见的教师画像

### 画像一：教学型教师

**典型特征**：
- 主要精力放在教学上
- 最烦恼的是备课时间不够
- 想花更多时间研究教学方法

**推荐起点**：第4章 备课助手

> "我教两个班数学，每天光写教案就两三个小时。如果AI能帮我搞定初稿，我就能把时间花在怎么教得更好上。" —— 张老师，初二数学

### 画像二：班主任型教师

**典型特征**：
- 既要教学又要管班级
- 活动多、琐事多
- 家校沟通压力大

**推荐起点**：第6章 活动助手 或 第8章 班主任助手

> "我是班主任，活动策划、家长沟通、学生管理全要管。每天下班都觉得没做完事。" —— 李老师，五年级班主任

### 画像三：行政型教师

**典型特征**：
- 主要做行政管理工作
- 报告、计划、总结写不完
- 需要处理大量数据和信息

**推荐起点**：第7章 行政助手

> "学期初和学期末最忙，各种计划总结报表堆在一起。希望能自动化一部分。" —— 王主任，教务主任

## 确定你的第一个智能体

完成测试后，你应该有了一个初步方向。接下来：

1. **确定第一个智能体的类型**（从上面的诊断表中选）
2. **写下你最希望它解决的 3 个具体问题**
3. **继续学习第2章，开始搭建**

> 💡 **建议**：第一个智能体不要贪多，解决一个具体痛点就好。做好一个之后，再做下一个。
