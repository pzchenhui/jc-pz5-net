# 2.4 能力编排：定制你的智能体

## 什么是能力编排？

人设定义了智能体的"性格"，但光有性格还不够——它需要**实际能力**来完成任务。

> 🎯 **能力编排**就是给智能体添加各种"技能"，让它能做得更多、更好。

## 三大能力模块

```
┌──────────────────────────────────────┐
│           智能体的能力模块             │
├────────────┬────────────┬────────────┤
│  📚 知识库  │  🔌 插件    │  🔄 工作流  │
│            │            │            │
│ 教它你的     │ 让它连接    │ 让它执行    │
│ 专属知识     │ 外部工具    │ 复杂流程    │
└────────────┴────────────┴────────────┘
```

## 📚 一、知识库——教它你的专属知识

### 为什么需要知识库？

通用AI知道全世界知识，但不知道：
- 你们学校的教学规范
- 你使用的教材版本
- 你的教学风格偏好
- 校本教研的成果

<div class="case-card">
<span class="case-label">📌 真实案例</span>

### 有知识库 vs 无知识库

**无知识库的AI**：
> 问："请根据课标设计七年级一元一次方程的教学目标"
> 答："教学目标：1. 理解方程的概念 2. 掌握解方程的方法..."
> （通用回答，不一定符合你们的课标版本）

**有知识库的AI**（上传了课标文件）：
> 问："请根据课标设计七年级一元一次方程的教学目标"
> 答："根据《义务教育数学课程标准（2022年版）》第三学段'数与代数'领域要求：
> '理解一元一次方程的概念，能根据具体问题列出方程，体会方程是刻画现实世界数量关系的有效模型。'
> 
> 教学目标：
> 1. 能说出方程和一元一次方程的定义（对应课标'理解概念'）
> 2. 能从实际问题中找出等量关系并列出方程（对应课标'体会方程是刻画现实世界的模型'）..."
> （引用了具体课标条目，更有依据）

</div>

### 支持的文件格式

| 格式 | 适合内容 | 示例 |
|------|---------|------|
| PDF | 课程标准、政策文件 | 义务教育课程标准.pdf |
| Word | 教案模板、教学规范 | 教案标准模板.docx |
| TXT | 纯文本资料 | 教学计划.txt |
| Excel | 数据表格 | 成绩分析模板.xlsx |
| 网页链接 | 在线资源 | 教材电子网址 |

### 📋 实操步骤：上传知识库

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">准备文件</span>

收集以下文件：
- 课程标准文件（PDF或Word）
- 教材目录或电子教材
- 学校教案模板
- 优秀教案范例
</div>

<div class="step-card">
<span class="step-number">2</span>
<span class="step-title">上传到Coze</span>

1. 在智能体配置页面找到 **"知识库"** 模块
2. 点击 **"添加知识库"**
3. 选择 **"上传文件"**
4. 选择准备好的文件
5. 等待文件处理完成（通常1-2分钟）
</div>

<div class="step-card">
<span class="step-number">3</span>
<span class="step-title">测试效果</span>

上传完成后，在预览窗口测试：
1. 问一个与知识库内容相关的问题
2. 观察回复是否引用了知识库内容
3. 对比上传前后的回答差异
</div>

## 🔌 二、插件——让它连接外部世界

### 插件是什么？

插件是智能体的**手脚**，让它能：上网搜索、生成图片、处理文档、进行计算、发送消息。

### 常用插件推荐

| 插件 | 用途 | 教师场景 |
|------|------|---------|
| **联网搜索** | 搜索互联网信息 | 查找最新教学资源、时事素材 |
| **图片生成** | 根据描述生成图片 | 制作课件插图、活动海报 |
| **文档阅读** | 读取上传的文档 | 读取教材、参考书内容 |
| **代码执行** | 运行代码片段 | 数学计算、数据处理 |

### 📋 实操步骤：添加插件

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">打开插件模块</span>

在智能体配置页面找到 **"插件"** 模块，点击 **"添加插件"**
</div>

<div class="step-card">
<span class="step-number">2</span>
<span class="step-title">选择需要的插件</span>

从插件市场中浏览并选择你需要的插件，点击 **"添加"**
</div>

<div class="step-card">
<span class="step-number">3</span>
<span class="step-title">测试插件功能</span>

在预览窗口测试插件是否正常工作，例如测试联网搜索：
> "帮我搜索一下2024年最新的数学教学案例"
</div>

## 🔄 三、工作流——让它执行复杂任务

工作流是更高级的能力编排方式，让智能体按**预设的流程**自动执行多步任务。

> 📖 工作流内容较多，将在第3章详细讲解。

## 能力编排的四个层次

| 层次 | 配置 | 适合谁 |
|------|------|--------|
| **L1 基础版** | 只有人设 | 简单问答场景 |
| **L2 标准版** | 人设 + 知识库 | 需要专业知识回答 |
| **L3 增强版** | 人设 + 知识库 + 插件 | 需要联网搜索、图片生成等 |
| **L4 高级版** | 人设 + 知识库 + 插件 + 工作流 | 需要复杂多步骤任务 |

> 💡 **建议**：从 L1 开始，逐步升级。不要一开始就配置所有能力。
