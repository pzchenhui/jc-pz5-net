# 3.2 工作流有何优势？

## 没有工作流 vs 有工作流

<div class="case-card">
<span class="case-label">📌 对比案例</span>

### 场景：准备一堂公开课的完整材料

**没有工作流**——需要手动操作7步：

1. 打开课标知识库 → 搜索对应知识点 → 3分钟
2. 复制课标要求 → 粘贴到对话框 → 1分钟
3. 让AI生成教案 → 等3分钟 → 人工审核 → 5分钟
4. 让AI基于教案出题 → 等2分钟 → 审核答案 → 5分钟
5. 让AI生成课件大纲 → 等1分钟 → 调整格式 → 5分钟
6. 让AI写评课表 → 等1分钟
7. 把所有材料整理到一个文档 → 10分钟

> **总耗时：约 33 分钟，需要7次交互操作**

**有工作流**——只需1步：

1. 输入课题 → 点击运行 → 等待 → 获得完整教学包

> **总耗时：约 5 分钟，只需1次操作**
> ⏱️ **效率提升约 6.6 倍！**

</div>

## 工作流的五大优势

### 1️⃣ 一键执行，减少重复操作

| 场景 | 手动操作次数 | 工作流操作次数 |
|------|-------------|---------------|
| 备课 | 7次 | 1次 |
| 出试卷 | 5次 | 1次 |
| 写周报 | 4次 | 1次 |

### 2️⃣ 流程标准化，保证质量一致性

手动操作时，每次的步骤可能不一样，导致输出质量参差不齐。

工作流确保**每次都按相同的步骤执行**，输出质量稳定。

<div class="template-card">
<span class="template-label">📝 标准化示例</span>

**备课工作流的固定步骤**：
1. 查询课标 → 2. 生成教案 → 3. 出练习题 → 4. 做课件大纲 → 5. 整合输出

无论输入什么课题，都严格按照这5步执行，不会遗漏任何环节。

</div>

### 3️⃣ 降低使用门槛

| 用户类型 | 手动操作 | 使用工作流 |
|----------|---------|-----------|
| 新手教师 | 需要知道每一步怎么做 | 只需输入课题 |
| 老教师 | 可能忘记某些步骤 | 工作流自动完成 |
| 行政人员 | 不熟悉教学流程 | 工作流内置流程 |

### 4️⃣ 可复用、可分享

工作流一旦搭建好，可以：
- 分享给同事直接使用
- 在不同学科间复制修改
- 作为学校的标准流程

### 5️⃣ 支持定时和自动触发

工作流可以设置为：
- ⏰ 每天定时执行（如生成每日教学提醒）
- 📧 收到邮件时自动触发
- 📅 每周一自动生成周报

## 工作流 vs 普通智能体：使用场景对比

| 场景 | 适合方式 | 原因 |
|------|---------|------|
| 回答一个知识问题 | 普通智能体 | 单步任务 |
| 生成一份教案 | 普通智能体 | 单步任务 |
| 一键准备完整教学包 | **工作流** | 多步骤，固定流程 |
| 批量生成学生评语 | **工作流** | 重复性多步骤任务 |
| 自动生成教学周报 | **工作流** | 定时执行 |
| 学生答疑 | 普通智能体 | 交互式对话 |

## 什么时候该搭建工作流？

当你发现自己在反复做以下事情时，就该考虑搭建工作流了：

| 信号 | 说明 |
|------|------|
| 每次都要做同样的3-5个步骤 | 适合工作流 |
| 每周/每月都要做同样的事 | 适合工作流+定时触发 |
| 不同的老师做同一件事步骤不同 | 适合工作流标准化 |
| 新老师不知道怎么做 | 适合工作流降低门槛 |
