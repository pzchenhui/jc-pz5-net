# 📝 第3章练习：工作流进阶

## 练习目标

搭建一个简单工作流，让智能体实现多步骤自动执行。

---

## 🛠️ 实操任务：搭建备课工作流

### 任务说明

搭建一个"一键备课"工作流：输入课题 → 查询课标 → 生成教案 → 生成练习题 → 输出完整教学包。

### 步骤检查清单

| 步骤 | 完成标准 | 打勾 |
|------|---------|------|
| 1. 创建工作流 | 在Coze中创建新工作流 | □ |
| 2. 设置开始节点 | 定义输入参数 | □ |
| 3. 添加知识库检索节点 | 连接课标知识库 | □ |
| 4. 添加教案生成节点 | 配置大模型节点 | □ |
| 5. 添加练习题生成节点 | 基于教案出题 | □ |
| 6. 添加结束节点 | 整合输出 | □ |
| 7. 测试运行 | 至少测试2个课题 | □ |
| 8. 发布并关联智能体 | 智能体能调用工作流 | □ |
