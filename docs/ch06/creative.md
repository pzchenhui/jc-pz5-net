# 6.3 添加并设计活动创意策划智能体

## 活动创意策划——从0到1

这是活动助手最核心的功能：帮你想出一个好活动方案。

<div class="case-card">
<span class="case-label">📌 真实案例</span>

### 赵老师的烦恼

赵老师是大队辅导员，每学期要策划10+场校园活动。

**以前策划一场活动**：
- 查往年方案找灵感 → 30分钟
- 结合今年情况调整 → 60分钟
- 写完整方案 → 90分钟
- 跟领导沟通修改 → 60分钟
- **合计：约 4 小时**

**用活动创意策划智能体**：
- 输入活动需求 → 2分钟
- AI生成3个方案 → 3分钟
- 选择一个并调整 → 20分钟
- **合计：约 25 分钟**

> ⏱️ **节省了 3.5 小时，效率提升约 9.6 倍！**

</div>

## 🛠️ 实操步骤

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">编写策划人设</span>

```markdown
你是一位校园活动策划专家，有10年中小学活动策划经验。
你了解不同年龄段学生的特点，擅长设计安全、有趣、有意义的校园活动。

## 策划方案必须包含：
- 活动背景与目标
- 活动对象与规模
- 活动时间与地点
- 活动流程（精确到每个环节的时间）
- 人员分工
- 物料清单与预算
- 安全预案
- 备选方案（天气等突发情况）

## 回复要求
- 活动设计考虑学生年龄特点
- 安全第一，户外活动必附安全注意事项
- 方案务实可执行
- 预算合理，适合学校经费实际
- 提供2-3个创意方案供选择
```
</div>

<div class="step-card">
<span class="step-number">2</span>
<span class="step-title">使用方式</span>

### 基础提问
```
学校要举办秋季运动会，帮我策划一个方案
```

### 精准提问（效果更好）
```
学校要举办秋季运动会，请策划方案：
- 对象：全校师生（约1500人）
- 时间：半天
- 地点：学校操场
- 要求：全员参与、趣味为主、安全第一
- 预算：5000元以内
```
</div>

<div class="step-card">
<span class="step-number">3</span>
<span class="step-title">配置知识库</span>

上传以下文件到知识库：

| 文件 | 用途 |
|------|------|
| 往年活动方案（2-3份） | 参考风格和格式 |
| 学校场地信息 | 了解可用场地条件 |
| 安全管理制度 | 确保方案合规 |
| 经费预算标准 | 控制预算合理 |
</div>

## 📋 案例输出

<div class="template-card">
<span class="template-label">📝 输出示例</span>

**输入**：
> 策划一场三年级的"秋天的发现"主题户外实践活动，2小时，在学校附近公园。

**输出**：

```
🍂 "秋天的发现"主题户外实践活动方案

一、活动目标
1. 观察秋天的自然变化，培养观察力
2. 通过小组合作，增强团队协作意识
3. 用语言表达发现，提升写作素材积累

二、活动流程
1. 集合整队（10分钟）
2. 安全教育（5分钟）
3. 自由观察收集（30分钟）
4. 小组分享（25分钟）
5. 创意手工（30分钟）
6. 展示与总结（15分钟）
7. 返程（5分钟）

三、物料清单
- 收集袋（每组1个）
- 手工材料（胶水、卡纸等）
- 急救包
- 签到表

四、安全预案
- 每组至少1名教师跟随
- 提前勘察场地
- 准备联系方式
- 雨天备选方案：教室室内活动
```
</div>

## 常用活动类型速查

| 活动类型 | 关注重点 | 提示词示例 |
|----------|---------|-----------|
| 运动会 | 安全、公平、全员参与 | "策划运动会方案，注重安全" |
| 文艺汇演 | 节目质量、舞台流程 | "文艺汇演流程安排" |
| 科技节 | 创新、动手、跨学科 | "科技节创意活动方案" |
| 读书活动 | 阅读激励、分享 | "读书月活动方案" |
| 社会实践 | 安全、教育价值 | "社会实践策划方案" |
| 主题班会 | 教育意义、参与 | "爱国主义教育主题班会" |
