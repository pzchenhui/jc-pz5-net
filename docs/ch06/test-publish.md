# 6.7 学科教学智能体测试与发布

## 全面测试清单

学科教学智能体的测试需要覆盖**学科准确性、教学有效性、输出规范性**三个维度。

### 学科准确性测试

| 测试类型 | 方法 | 通过标准 |
|----------|------|---------|
| 知识问答 | 问5个学科知识点 | 回答100%准确 |
| 题目生成 | 生成3道练习题 | 题目正确、答案无误 |
| 解题指导 | 提供2道题目 | 解题逻辑正确、步骤清晰 |
| 批改评价 | 上传3份学生作业 | 批改准确、评价合理 |

### 教学有效性测试

| 测试项 | 方法 | 通过标准 |
|--------|------|---------|
| 难度适配 | 同一内容测试小学/初中/高中 | 输出符合学段要求 |
| 教学方法 | 检查教学方法是否恰当 | 符合学科教学规律 |
| 引导性 | 检查是否引导学生思考 | 不直接给答案（数学） |
| 规范性 | 检查输出格式是否规范 | 格式统一、结构清晰 |

### 输出规范性测试

| 测试项 | 方法 | 通过标准 |
|--------|------|---------|
| 格式规范 | 检查输出是否符合模板 | ≥90%符合 |
| 评分标准 | AI评分与人工评分对比 | 误差≤5分（百分制） |
| 语言规范 | 检查用词用语 | 专业、得体 |
| 安全合规 | 检查内容是否合规 | 无不当内容 |

## 发布渠道推荐

| 渠道 | 适合场景 | 配置方式 |
|------|---------|---------|
| 教师工作群 | 学科教研组使用 | 分享链接到工作群 |
| 学校网站嵌入 | 全校教师使用 | 嵌入式网页 |
| 教师个人使用 | 个人备课辅助 | 单独链接 |
