# 6.8 巧用AI工具：学科教学全流程工具链

## 学科教学全流程——AI工具矩阵

学科教学不只是备课，还包括**上课、批改、评价、反馈**等多个环节。每个环节都有适合AI辅助的场景。关键是**找对工具、用对场景、控制质量**。

## 语文学科全流程

| 环节 | AI工具 | 具体应用 | 推荐平台 | 操作要点 |
|------|--------|---------|---------|---------|
| 备课 | 教案生成+课件制作 | 根据课标和教材生成教案和课件框架 | Coze | 上传课程标准+教材目录 |
| 上课 | 课堂互动设计 | 设计讨论题、活动环节、情境导入 | 智谱清言 | 输入课文+教学目标 |
| 作业 | 作文批改 | AI初评（内容/结构/语言/书写）+人工复核 | Coze | 上传评分细则到知识库 |
| 评价 | 阅读能力分析 | 根据阅读理解答题数据分析薄弱点 | Kimi | 上传答题数据+分析维度 |
| 反馈 | 家长沟通话术 | 生成个性化学习情况反馈 | 智谱清言 | 输入学生表现关键词 |

## 跨学科通用工具

| 工具 | 用途 | 推荐平台 | 使用场景 |

## 工具选择决策树

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">明确需求</span>
- 你是要生成内容（教案/题目）？→ 选内容生成类
- 还是要分析数据（成绩/错题）？→ 选数据分析类
- 还是要对话交互（口语练习）？→ 选对话类

<span class="step-number">2</span>
<span class="step-title">评估复杂度</span>
- 简单任务（单次问答）：智谱清言、Kimi、通义千问
- 复杂任务（多步工作流）：Coze
- 需要自建部署：Dify

<span class="step-number">3</span>
<span class="step-title">选择平台</span>
- 语文学科 → 智谱清言（中文理解最强）
- 数学学科 → Coze（工作流编排）+ 通义千问（计算验证）
- 英语学科 → Coze（多模态+对话）+ 通义千问（语音合成）
- 科学/物理 → Coze（实验设计工作流）+ 智谱清言（数据分析）
- 历史/地理 → Kimi（长文档分析史料/地图）

<span class="step-number">4</span>
<span class="step-title">测试验证</span>
- 用典型任务测试3-5次
- 检查输出质量是否稳定
- 确认无误后再正式使用
</div>

## 工具组合推荐方案

### 方案A：轻量级（零代码，适合个人教师）

| 场景 | 工具 | 说明 |
|------|------|------|
| 日常备课 | 智谱清言/Kimi | 输入教学目标，生成教案框架 |
| 题目生成 | 智谱清言 | 输入知识点，生成练习题 |
| 作文批改 | Coze智能体 | 配置评分细则，拍照上传 |
| 数据分析 | Kimi | 上传Excel成绩表，自动生成分析报告 |

### 方案B：专业级（工作流编排，适合教研组）

| 场景 | 工具 | 说明 |
|------|------|------|
| 智能体搭建 | Coze | 搭建学科教学智能体（人设+知识库+工作流） |
| 长文档分析 | Kimi | 课标解读、教材分析、试卷分析 |
| 语音合成 | 通义千问 | 听力材料语音生成 |
| 数据分析 | Coze+Kimi | Coze处理结构化数据，Kimi分析非结构化数据 |

### 方案C：自研级（自建部署，适合学校/区域）

| 场景 | 工具 | 说明 |
|------|------|------|
| 平台搭建 | Dify | 自建AI平台，集成多种模型 |
| 数据管理 | 自建数据库 | 学生数据、题库、教学资源 |
| 智能体开发 | Dify+API | 自建学科教学智能体 |
| 系统集成 | API对接 | 与学校现有系统（教务、成绩）对接 |
