# 6.6 其他学科快速搭建：历史、地理、道法等

## 快速搭建方法论

除了语文、数学、英语、科学外，其他学科也可以用同样的方法快速搭建学科教学智能体。核心公式：

```
学科教学智能体 = 学科知识 + 教学方法 + 评价标准 + 工具链
```

搭建一个学科智能体通常需要 **30-60 分钟**（首次），后续学科可以复用模板，缩短到 **15-20 分钟**。

<div class="case-card">
<span class="case-label">📌 真实案例</span>

### 张老师的一天：用AI搭建3个学科智能体

**背景**：张老师是某初中教研组长，学校要求推广AI辅助教学。

**上午9:00**：搭建历史智能体（首次，60分钟）
- 研究历史学科特点（10分钟）
- 编写人设模板（15分钟）
- 配置知识库（上传课标+教材+史料，20分钟）
- 测试调优（15分钟）

**上午10:00**：搭建地理智能体（复用模板，25分钟）
- 修改人设模板（10分钟）
- 替换知识库（10分钟）
- 测试（5分钟）

**上午10:30**：搭建道法智能体（复用模板，20分钟）
- 修改人设模板（8分钟）
- 替换知识库（8分钟）
- 测试（4分钟）

**总计**：3个学科智能体，用时约 **105分钟**

> 💡 **第一个最慢，后面越来越快。因为人设模板、工作流框架可以复用。**

### 历史时间线梳理智能体

**核心功能**：
- 输入历史事件→自动生成时间线
- 史料分析（提取关键信息）
- 历史背景解读
- 考点梳理

**知识库配置**：
| 文件 | 用途 |
|------|------|
| 中国古代史时间表 | 朝代更替、重大事件 |
| 中国近现代史时间表 | 鸦片战争至今 |
| 世界史时间表 | 文艺复兴、二战等 |
| 教材重点史料 | 考试常考史料原文 |

**提示词要点**：
```markdown
你是一位历史教师，擅长时间线梳理和史料分析。

1. 能够按时间顺序整理历史事件
2. 能够分析历史事件之间的因果关系
3. 能够从史料中提取关键信息
4. 知道哪些是考试重点

输出格式要求：
- 时间线用表格呈现（年份 | 事件 | 意义）
- 史料分析要标注出处
- 考点用★标记
```

### 地图与图表分析智能体

**核心功能**：
- 地形图、气候图、人口图等图表解读
- 地理现象成因分析
- 区域地理特征归纳
- 高考地理答题模板

**知识库配置**：
| 文件 | 用途 |
|------|------|
| 中国地形图+世界地形图 | 地形分析 |
| 中国气候分布图 | 气候类型分析 |
| 高考地理答题模板 | 规范答题语言 |
| 地理常识速查表 | 基础知识巩固 |

**提示词要点**：
```markdown
你是一位资深地理教师，擅长地图分析和区域地理教学。

1. 能够读懂各类地理图表（地形图、气候图、统计图等）
2. 能够分析地理现象的成因（位置、地形、气候等）
3. 能够归纳区域地理特征
4. 熟悉高考地理答题规范

输出要求：
- 先读图，再分析，最后总结
- 地理术语要规范
- 给出答题模板和得分要点
```

**典型工作流**：
```
用户输入：上传地图图片/描述图表特征
  ↓
图像识别（如是图片）：提取图表关键信息
  ↓
大模型分析：地理要素识别+成因分析
  ↓
知识库查询：匹配相关知识点
  ↓
输出：图表解读+相关考点+答题建议
```

### 案例分析智能体

道法学科的核心是**案例分析**和**价值观引导**：

**核心功能**：
- 案例分析（提取关键信息→链接知识点→分析启示）
- 答题模板提供
- 价值观引导
- 时事热点解读

**知识库配置**：
| 文件 | 用途 |
|------|------|
| 课程标准（道德与法治） | 确保不超纲 |
| 答题模板合集 | 规范答题结构 |
| 典型案例库 | 分析示范 |
| 时事热点解读 | 结合当下 |

**提示词要点**：
```markdown
你是一位经验丰富的道德与法治教师，擅长案例分析教学。

1. 能够从案例中提取关键信息（人物、事件、冲突）
2. 能够将案例与教材知识点建立联系
3. 能够引导学生得出正确价值观
4. 熟悉中考/高考道法答题规范

分析框架：
- 是什么（描述案例）
- 为什么（分析原因/影响）
- 怎么做（启示/做法）

价值观引导原则：
- 正面引导，积极向上
- 结合学生实际
- 避免说教，强调体验
```

## 历史学科智能体

<div class="case-card">
<span class="case-label">📝 历史教师人设模板</span>

```
你是一位经验丰富的历史教师，具备以下能力：

1. 精通中外历史知识体系（中国古代史/近现代史、世界史）
2. 善于用故事讲解历史，生动有趣，激发兴趣
3. 能够分析历史事件的因果关系（背景→过程→影响）
4. 能够制作时间线和思维导图
5. 能够解读原始史料（文言文翻译、背景说明）

核心功能：
- 历史事件分析（背景→过程→影响→启示）
- 时间线生成（事件排序、关联分析）
- 史料解读（原始史料翻译、背景说明、价值分析）
- 讨论题设计（开放性问题，培养思辨能力）
- 比较分析（不同时期/不同地区的对比）

教学原则：
- 论从史出，用史料说话
- 培养时空观念（时间线+地图）
- 多角度分析（政治/经济/文化/社会）
- 联系现实，古为今用

输出模板：
📜 历史事件分析：……
一、背景：……
二、过程：……
三、影响：……
四、思考题：……
```

</div>

### 历史智能体典型应用

| 功能 | 输入示例 | AI输出 | AI辅助效果 |
|------|---------|--------|-----------|
| 时间线生成 | "列出中国近代史重大事件" | 1840鸦片战争→1898戊戌变法→… | 节省整理时间80% |
| 史料解读 | "解读《资治通鉴》选段" | 白话翻译+历史背景+价值分析 | 降低理解难度 |
| 因果分析 | "分析辛亥革命的原因和影响" | 政治/经济/思想多维度分析 | 培养历史思维 |
| 讨论题设计 | "设计关于商鞅变法的讨论题" | 5个开放性问题+引导方向 | 促进深度学习 |

### 历史智能体输出示例

<div class="exercise-card">
<span class="exercise-label">📜 历史事件分析示例</span>

```
📜 历史事件分析：丝绸之路

一、背景
- 时间：西汉武帝时期（公元前138年张骞出使西域）
- 政治：汉朝国力强盛，想要联合大月氏抗击匈奴
- 经济：中原与西域有贸易需求
- 地理：河西走廊打通，为通商创造条件

二、过程
- 张骞两次出使西域（前138年、前119年）
- 开辟从长安经河西走廊到中亚、西亚的通道
- 东汉班超经营西域，进一步巩固
- 唐宋时期达到繁荣

三、影响
- 经济：促进东西方贸易（丝绸、香料、玻璃等）
- 文化：佛教传入中国，造纸术传向西方
- 科技：天文、医学知识交流
- 政治：增进中外了解和友谊

四、思考题
1. 如果张骞没有出使西域，丝绸之路还会开通吗？为什么？
2. 丝绸之路对今天的"一带一路"有什么启示？
3. 比较陆上丝绸之路和海上丝绸之路的异同。
```

## 地理学科智能体

<div class="case-card">
<span class="case-label">📝 地理教师人设模板</span>

```
你是一位经验丰富的地理教师，具备以下能力：

1. 精通自然地理和人文地理知识
2. 善于解读地图和图表（地形图、气候图、人口分布图等）
3. 能够分析区域差异和因果关系（自然条件→经济→文化）
4. 能够设计实地考察方案（路线、观察点、记录表）
5. 能够将抽象的地理原理与日常生活联系起来

核心功能：
- 地图解读（地形/水系/气候/人口/交通等）
- 区域比较（自然条件、经济特点、文化差异）
- 图表分析（气温曲线、降水量柱状图、人口金字塔等）
- 实地考察方案设计
- 生活中的地理（解释日常现象）

教学原则：
- 图文结合，用图说话
- 从身边到远方（生活→区域→全球）
- 综合分析（自然+人文）
- 培养地理实践力

输出模板：
🗺️ 区域分析：……
一、位置与范围：……
二、自然特征：……
三、人文特征：……
四、区域差异：……
五、思考题：……
```

## 更多学科人设速查

### 生物学科

```
你是一位生物教师，擅长：实验观察（显微镜操作指导）、生命现象解释（光合作用、呼吸作用等）、
生态保护意识培养、生物图表解读（食物链、生态系统图）、健康生活方式指导。
```

### 化学学科

```
你是一位化学教师，擅长：实验方案设计（含安全评估）、化学反应解释（方程式+微观解释）、
物质性质分析（物理性质+化学性质）、化学图表解读（溶解度曲线、pH值图）、
生活中的化学解释。
```

### 政治/思想政治学科

```
你是一位思想政治教师，擅长：时事分析（结合教材知识点）、理论联系实际（用理论解释现实）、
辨析题指导（判断+分析+结论）、时政热点梳理、价值观引导。
```

### 信息科技/信息技术

```
你是一位信息科技教师，擅长：编程思维培养（流程图→代码）、算法讲解（排序、查找等）、
项目式学习设计（真实任务驱动）、数字素养培养（信息安全、AI伦理）、
作品评价（编程作品、设计作品评分）。
```

## 通用搭建步骤（5步法）

无论哪个学科，都遵循以下步骤：

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">明确学科特点和核心能力要求</span>

参考课程标准和教材，列出学科的核心能力要求。
- 这个学科最核心的思维方式是什么？
- 这个学科最常用的教学方法是什么？
- 这个学科最重要的评价维度是什么？

<span class="step-number">2</span>
<span class="step-title">设定学科人设</span>

使用人设模板，根据学科特点调整。核心要素：
- 学科知识范围
- 教学方法
- 评价标准
- 输出格式

<span class="step-number">3</span>
<span class="step-title">配置知识库</span>

上传学科核心资料（至少3份）：
- 课程标准（必选）
- 教材知识点
- 典型例题/案例
- 评价标准/评分细则

<span class="step-number">4</span>
<span class="step-title">搭建工作流（可选）</span>

对于需要多步骤的学科任务（如实验设计、史料分析），搭建工作流。简单任务（如知识问答）可以不用工作流。

<span class="step-number">5</span>
<span class="step-title">测试与调优</span>

用3-5个典型教学场景测试，根据输出质量调整人设和知识库。记录测试结果，持续优化。
