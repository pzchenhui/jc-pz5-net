# 6.5 科学/物理学科智能体：实验设计+数据分析

## 科学教学——最需要实践指导的学科

科学/物理教学的特点是**实验驱动、数据说话、逻辑推理**。AI在实验设计和数据分析方面可以发挥重要作用，尤其在**安全评估**和**方案优化**环节。

<div class="case-card">
<span class="case-label">📌 真实案例</span>

### 刘老师的实验方案设计优化

刘老师是某初中物理教师，每学期需要设计10+个实验方案。

**传统方式**：
1. 查阅教材和教辅→设计实验步骤→评估安全性→准备材料清单
2. 每个实验方案：约1-2小时
3. 安全性评估靠经验，有时需要反复修改

**使用AI科学智能体后**：
1. 输入实验目的→AI推荐实验方案→自动评估安全性→生成材料清单
2. 每个实验方案：约15分钟（含人工复核）

> ⏱️ **节省了约1.5小时/次，效率提升约6倍。而且安全性评估更系统。**

## 实验设计智能体搭建

### Step 1：设定人设

<div class="template-card">
<span class="template-label">📝 科学教师人设模板</span>

```
你是一位经验丰富的中小学科学/物理教师，具备以下能力：

1. 精通实验设计和操作流程
2. 熟悉科学探究方法（观察、假设、实验、分析、结论）
3. 善于引导学生通过实验发现规律
4. 能够评估实验安全性和可行性

实验设计流程：
- 第一步：明确实验目的（验证什么规律/原理）
- 第二步：选择实验方法（演示实验/分组实验/探究实验）
- 第三步：设计实验步骤（清晰、可操作）
- 第四步：准备材料清单（易获取、低成本）
- 第五步：安全评估（风险评估+应急预案）
- 第六步：数据记录表格设计
- 第七步：思考题设计（引导学生分析数据、得出结论）

教学原则：
- 安全第一，所有实验必须通过安全评估
- 低成本优先，尽量使用常见材料
- 探究导向，让学生自己发现规律
- 数据驱动，用数据说话
```

</div>

### Step 2：实验方案输出模板

<div class="template-card">
<span class="template-label">📝 实验方案输出模板</span>

```
🔬 实验方案：探究浮力大小与排开液体体积的关系

一、实验目的
- 验证阿基米德原理：浮力大小等于物体排开液体所受的重力

二、实验类型
- 探究实验（学生分组，4人/组）

三、实验材料（每组）
- 弹簧测力计 × 1个
- 烧杯（500ml）× 1个
- 小石块 × 1个
- 细线 × 1根
- 水 × 适量
- 盐水（可选）× 适量

四、实验步骤
1. 用弹簧测力计测出小石块的重力G
2. 将烧杯装满水，放在桌面上
3. 用细线系住小石块，挂在弹簧测力计下
4. 将小石块缓慢浸入水中，记录测力计示数F
5. 计算浮力F浮 = G - F
6. 重复实验3次，取平均值

五、安全评估
- 风险等级：低（绿色）
- 注意事项：
  1. 水不要洒在桌面上，防止滑倒
  2. 弹簧测力计不要超量程使用
  3. 实验后及时清理桌面

六、数据记录表
| 实验次数 | G(N) | F(N) | F浮(N) |
|---------|------|------|--------|
| 1       |      |      |        |
| 2       |      |      |        |
| 3       |      |      |        |
| 平均值   |      |      |        |

七、思考题
1. 浮力大小与什么因素有关？
2. 如果把水换成盐水，浮力会怎样变化？为什么？
3. 这个实验能说明什么物理原理？
```

## 数据分析智能体

### 人设模板

<div class="template-card">
<span class="template-label">📝 数据分析人设</span>

```
你是一位科学数据分析专家，具备以下能力：

1. 能够处理实验数据（计算平均值、标准差等）
2. 能够绘制图表（折线图、柱状图、散点图）
3. 能够分析数据趋势和规律
4. 能够判断实验结论是否成立
5. 能够进行误差分析

分析流程：
- 第一步：整理数据（检查异常值，剔除明显错误数据）
- 第二步：计算统计量（平均值、最大值、最小值、标准差）
- 第三步：绘制图表（选择合适的图表类型）
- 第四步：分析趋势（数据说明了什么规律）
- 第五步：得出结论（结论是否支持假设）
- 第六步：误差分析（可能的误差来源及改进建议）
```

</div>

### 数据分析输出示例

<div class="exercise-card">
<span class="exercise-label">📊 数据分析输出示例</span>

```
📈 实验数据分析报告

实验：探究弹簧伸长量与拉力的关系

一、原始数据
| 拉力F(N) | 0 | 1 | 2 | 3 | 4 | 5 |
|---------|---|---|---|---|---|---|
| 伸长量x(cm) | 0 | 2.1 | 4.0 | 5.8 | 8.1 | 9.9 |

二、数据处理
- 计算F/x比值：2.1, 2.0, 1.93, 2.03, 1.98
- 平均比值：2.01 N/cm
- 标准差：0.06 N/cm

三、图表建议
- 散点图：F-x关系图（横轴x，纵轴F）
- 观察：数据点近似在一条直线上

四、结论
- 在弹性限度内，弹簧伸长量与拉力成正比
- 弹簧劲度系数 k ≈ 2.01 N/cm

五、误差分析
- 可能误差来源：
  1. 刻度尺读数误差（±0.1cm）
  2. 弹簧自身重力影响
  3. 弹簧未完全垂直
- 改进建议：
  1. 使用更精确的测量工具（游标卡尺）
  2. 多次测量取平均值
  3. 确保弹簧垂直悬挂
```

### 实验风险等级分类

| 等级 | 颜色 | 定义 | 示例 | 处理方式 |
|------|------|------|------|---------|
| 绿色 | 🟢 | 低风险 | 浮力实验、光的折射 | 正常进行 |

| 红色 | 🔴 | 高风险 | 有毒气体、爆炸实验 | **不建议在课堂进行** |

### 安全评估清单

AI自动检查以下项目：
- [ ] 是否使用危险化学品？（名称、浓度、用量）
- [ ] 是否涉及高温/高压/高速？
- [ ] 是否产生有害气体/烟雾？
- [ ] 是否有触电风险？
- [ ] 是否有尖锐物品？
- [ ] 学生是否需要防护装备？（护目镜、手套）
- [ ] 是否有应急预案？（洗眼器、灭火器位置）
