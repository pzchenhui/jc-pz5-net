# 📝 第6章练习：学科教学智能体

## 练习目标

完成一个学科教学智能体的搭建，并测试其教学效果。最终产出：**一个可实际使用的学科教学智能体**。

---

## 🛠️ 实操任务：搭建你的学科教学智能体

### 步骤检查清单

| 步骤 | 完成标准 | 预计用时 | 打勾 |
|------|---------|---------|------|
| 1. 选择学科 | 确定你要搭建的学科和学段 | 2分钟 | □ |
| 2. 设定人设 | 编写完整的学科教师人设 | 10分钟 | □ |
| 3. 配置知识库 | 上传至少3份学科核心资料 | 15分钟 | □ |
| 4. 搭建工作流 | 设计至少3个节点的教学工作流 | 15分钟 | □ |
| 5. 测试输出 | 用3种典型场景测试输出质量 | 10分钟 | □ |
| 6. 调优 | 根据测试结果调整人设/知识库/工作流 | 10分钟 | □ |

**总用时：约62分钟**

---

## 📋 练习一：学科人设编写（15分钟）

**根据你选择的学科，编写完整的人设：**

<div class="exercise-card">
<span class="exercise-label">📝 人设编写模板</span>

```
你是一位经验丰富的【学科】【学段】教师，具备以下能力：

1. 精通【学科】专业知识（列举3-5个核心知识领域）
2. 熟悉【学段】学生的认知特点和学习规律
3. 善于【学科特有的教学方法，如实验探究/情境教学等】
4. 能够根据学生水平调整教学策略

教学流程：
- 第一步：……（如：明确教学目标）
- 第二步：……（如：分析学情）
- 第三步：……（如：设计教学活动）
- 第四步：……（如：评价学习效果）

教学原则：
- 原则1：……
- 原则2：……
- 原则3：……
```

**完成后，请用以下问题自检：**
- [ ] 人设是否体现了学科特点？（如数学的逻辑性、语文的人文性）
- [ ] 教学流程是否完整？（备课→上课→评价）
- [ ] 教学原则是否符合课标要求？
- [ ] 语言是否专业、简洁、可操作？

</div>

## 📋 练习二：工作流设计（20分钟）

**设计一个学科教学工作流：**

<div class="exercise-card">
<span class="exercise-label">🔄 工作流设计模板</span>

| 节点 | 功能说明 | 输入内容 | 输出内容 | 失败处理 |
|------|---------|---------|---------|---------|
| 节点1 | 如：分析教学目标 | 课程标准+教材 | 教学目标清单 | 提示输入 |
| 节点2 | 如：设计教学活动 | 教学目标+学情 | 活动方案 | 重新生成 |
| 节点3 | 如：生成练习题 | 知识点+难度 | 练习题+答案 | 重新生成 |
| 节点4 | 如：生成评价标准 | 教学目标 | 评价量规 | 重新生成 |
| 节点5 | 如：格式化输出 | 以上所有 | 完整教案 | 重新生成 |

**设计原则：**
1. 每个节点的输入输出要明确
2. 节点之间有逻辑顺序
3. 考虑失败情况（如输入不完整时的处理）
4. 最终输出要格式统一、结构清晰

</div>

## 📋 练习三：测试与调优（15分钟）

**用以下场景测试你的学科教学智能体：**

<div class="exercise-card">
<span class="exercise-label">🧪 测试记录表</span>

| 测试项 | 输入内容 | 期望结果 | 实际结果 | 是否通过 | 改进措施 |
|--------|---------|---------|---------|---------|---------|
| 知识问答 | 问一个本学科的核心知识点 | 回答准确、有深度 | | □ | |
| 题目生成 | 指定知识点和难度生成1道题 | 题目正确、难度适中 | | □ | |
| 解题指导 | 提供一道典型题目 | 引导正确、步骤清晰 | | □ | |
| 批改评价 | 上传/输入一份学生答案 | 批改准确、有建议 | | □ | |
| 难度适配 | 同一内容测试不同学段 | 输出符合学段 | | □ | |

**调优建议：**
- 如果输出不准确 → 补充知识库
- 如果格式不规范 → 调整人设中的输出要求
- 如果引导不够好 → 优化工作流中的提示词
- 如果难度不适配 → 在知识库中添加学段区分

</div>

## 📋 练习四：跨学科思考（10分钟）

<div class="exercise-card">
<span class="exercise-label">💭 思考题</span>

1. **最适合AI辅助的环节**：你的学科教学中，哪个环节最适合用AI辅助？为什么？请用具体例子说明。

2. **分享给同科组老师**：如果要将你的AI智能体分享给同科组老师，需要做哪些调整？（考虑不同老师的教学风格差异）

3. **学科差异**：你的学科与其他学科在AI辅助方面有什么不同？请列举2-3个差异点。

4. **持续改进**：使用一个月后，你会收集哪些数据来评估这个智能体的效果？

**参考回答方向：**
- 第1题：可以思考批改、备课、出题等环节
- 第2题：考虑个性化配置（教学风格、评价标准）
- 第3题：不同学科在AI辅助上有不同侧重点
- 第4题：考虑准确性、效率、满意度等指标

</div>
