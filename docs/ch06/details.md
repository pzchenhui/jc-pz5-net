## 知识库配置

| 文件 | 用途 |
|------|------|
| 往年活动方案（2-3份） | 参考风格和格式 |
| 学校场地信息 | 了解可用场地条件 |
| 安全管理制度 | 确保方案合规 |
| 经费预算标准 | 控制预算合理 |
| 学校简介 | 提供学校基本信息 |

## 插件配置

| 插件 | 用途 |
|------|------|
| 联网搜索 | 查找最新活动创意和案例 |
| 图片生成 | 生成海报配图 |

## 多轮对话配置

- **开启对话记忆**：支持活动方案的逐步完善
- **对话轮数**：建议 20-30 轮（策划活动需要较多交互）
