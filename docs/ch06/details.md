## 知识库配置建议

| 文件 | 用途 | 推荐数量 |
|------|------|---------|
| 往年活动方案（2-3份） | 参考风格和格式 | 2-3份 |
| 学校场地信息 | 了解可用场地条件 | 1份 |
| 安全管理制度 | 确保方案合规 | 1份 |
| 经费预算标准 | 控制预算合理 | 1份 |
| 学校简介 | 提供学校基本信息 | 1份 |

## 插件配置

| 插件 | 用途 | 推荐度 |
|------|------|--------|
| 联网搜索 | 查找最新活动创意和案例 | ⭐⭐⭐⭐⭐ |
| 图片生成 | 生成海报配图 | ⭐⭐⭐⭐ |
| 文档处理 | 生成活动方案文档 | ⭐⭐⭐ |

## 多轮对话配置

| 设置项 | 建议值 | 说明 |
|--------|--------|------|
| 对话轮数 | 20-30轮 | 策划活动需要较多交互 |
| 开启对话记忆 | ✅ 开启 | 支持方案逐步完善 |
| 话题引导 | 开启 | 引导用户补充关键信息 |

## 用户体验优化

<div class="case-card">
<span class="case-label">💡 优化建议</span>

### 让活动助手更"懂学校"

1. **在知识库中上传**：
   - 学校校历（了解学期时间节点）
   - 活动审批流程（确保方案可执行）
   - 往年活动照片（参考风格）

2. **在人设中明确**：
   - 学校类型（小学/初中/高中）
   - 学生年龄段（影响活动设计）
   - 学校特色（体育强校/艺术特色等）

3. **在开场白中体现学校特色**：
   - "你好！我是XX小学的活动助手..."

</div>
