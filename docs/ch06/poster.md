## 活动海报的AI生成流程

<div class="case-card">
<span class="case-label">📌 对比案例</span>

### 传统海报制作 vs AI辅助

**传统流程**：
1. 想文案 → 30分钟
2. 找设计师或自己设计 → 2小时
3. 反复修改 → 1小时
4. 定稿出图 → 30分钟
5. **合计：约 4 小时**

**AI辅助流程**：
1. 输入活动信息 → 2分钟
2. AI生成海报文案 → 1分钟
3. AI生成设计方案 → 2分钟
4. 手动微调 → 10分钟
5. **合计：约 15 分钟**

> ⏱️ **节省了 3.75 小时，效率提升约 16 倍！**

</div>

## 📝 海报文案生成智能体

<div class="template-card">
<span class="template-label">📝 海报文案人设模板</span>

```markdown
你是一位校园宣传文案专家，擅长为各类校园活动
撰写吸引人的海报文案。

## 任务
根据活动信息，生成：
1. 主标题（简短有力，10字以内）
2. 副标题（补充说明，20字以内）
3. 活动亮点（3-5条，每条不超过15字）
4. 时间地点信息
5. 号召性用语

## 风格要求
- 面向学生：活泼有趣，用emoji
- 面向家长：正式温暖，用词得体
- 面向教师：简洁专业，重点突出

## 不同活动类型的标题风格
- 运动会：动感、竞技、团结
- 文艺汇演：文艺、创意、青春
- 科技节：科技、创新、未来
- 读书活动：书香、成长、智慧
- 主题班会：教育、温暖、力量
```
</div>

## 海报设计原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 少即是多 | 文字不要太多 | 标题突出，正文精简 |
| 色彩统一 | 主色调不超过3种 | 科技节用蓝+白+银 |
| 重点突出 | 最重要的信息最大 | 活动名称和时间最大 |
| 留白呼吸 | 不要挤满整个画面 | 适当留白更专业 |
