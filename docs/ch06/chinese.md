# 6.2 语文学科智能体：作文批改+阅读理解

## 语文教学——最需要AI辅助的学科

语文是**批改量最大、主观性最强**的学科：

<div class="case-card">
<span class="case-label">📌 真实案例</span>

### 张老师的作文批改困境

张老师是某初中语文教师，每班45人，两个班共90人。

**传统方式**：
1. 每次作文作业：逐字阅读→圈画错误→写评语→打分
2. 每份作文平均耗时：5-8分钟
3. 90份 × 6分钟 = **540分钟（9小时）**

**使用AI作文批改智能体后**：
1. 拍照/上传作文→AI初评（错字、病句、结构、立意）→人工复核
2. 每份复核时间：2分钟
3. 90份 × 2分钟 = **180分钟（3小时）**

> ⏱️ **节省了6小时，效率提升约3倍。而且AI的评语更规范、标准更一致。**

## 作文批改智能体搭建

### Step 1：设定人设

<div class="template-card">
<span class="template-label">📝 语文教师人设模板</span>

```
你是一位经验丰富的中小学语文教师，具备以下能力：

1. 精通作文批改（记叙文、议论文、说明文、应用文）
2. 熟悉语文课程标准中的写作要求
3. 善于发现学生的闪光点并提出改进建议
4. 能够根据学段调整评价标准

批改流程：
- 第一步：通读全文，判断文体
- 第二步：检查基础（错别字、病句、标点）
- 第三步：评价内容（主题、立意、选材）
- 第四步：评价结构（开头、过渡、结尾）
- 第五步：评价语言（表达、修辞、文采）
- 第六步：给出总评和改进建议

评价标准：
- 基础（30%）：错别字、病句、标点使用
- 内容（30%）：主题明确、立意深刻、选材恰当
- 结构（20%）：层次分明、过渡自然、首尾呼应
- 语言（20%）：表达流畅、修辞得当、有文采
```

</div>

### Step 2：配置知识库

| 文件 | 用途 | 推荐数量 |
|------|------|---------|
| 语文课程标准（写作部分） | 提供评价依据 | 1份 |
| 优秀作文范例 | 参考标准 | 5-10篇 |
| 常见错别字表 | 批改参考 | 1份 |
| 作文评分细则 | 统一评分标准 | 1份 |

### Step 3：配置工作流

<div class="case-card">
<span class="case-label">📝 作文批改工作流</span>

```
开始（上传作文）
  ↓
节点1：文体识别
  → 判断是记叙文/议论文/说明文/其他
  ↓
节点2：基础检查
  → 错别字、病句、标点错误
  → 输出：错误清单
  ↓
节点3：内容评价
  → 主题是否明确？立意是否深刻？
  → 输出：内容评分+评语
  ↓
节点4：结构评价
  → 层次是否分明？过渡是否自然？
  → 输出：结构评分+评语
  ↓
节点5：语言评价
  → 表达是否流畅？修辞是否得当？
  → 输出：语言评分+评语
  ↓
节点6：总评生成
  → 综合四项评分，生成总评
  → 输出：总分+总评+改进建议
  ↓
结束
```

</div>

## 阅读理解分析智能体

### 人设模板

<div class="template-card">
<span class="template-label">📝 阅读理解分析人设</span>

```
你是一位语文阅读理解辅导教师，具备以下能力：

1. 能够分析文章的主旨和中心思想
2. 能够解读关键词句的含义和作用
3. 能够分析段落结构和行文思路
4. 能够鉴赏修辞手法和表达技巧
5. 能够设计阅读理解题目

工作方式：
- 先整体把握文章内容和结构
- 再深入分析关键词句和修辞
- 最后提供阅读理解题目和答案
```

</div>

### 典型输出格式

```
📖 阅读理解分析

一、文章概览
- 文体：记叙文
- 主旨：通过...表达了...
- 情感：...

二、重点句段分析
1. "……"（第X段）
   → 含义：……
   → 作用：……
   → 修辞：……

三、阅读理解题
1. 选择题：……（答案：X）
2. 简答题：……（答案要点：……）
3. 开放题：……（参考思路：……）
```

