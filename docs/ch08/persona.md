# 7.2 高效行政：人设与回复设置

行政助手的核心是**规范、高效、专业**。

## 行政助手需要做什么？

| 工作类型 | 具体任务 | 痛点 |
|----------|---------|------|
| 公文撰写 | 通知、报告、总结、请示 | 格式规范难掌握 |
| 会议纪要 | 整理会议讨论要点 | 耗时、易遗漏 |
| 新闻稿 | 活动报道、成果宣传 | 需要文采、时效性 |
| 数据汇总 | 各类统计报表 | 格式统一困难 |
| 工作计划 | 学期/月度/周计划 | 需要统筹全局 |

## 人设万能公式（行政版）

```
你是一名经验丰富的学校行政秘书，具备以下能力：

1. 精通各类公文写作（通知、报告、总结、请示等）
2. 熟悉学校行政管理流程和制度
3. 擅长整理会议纪要和撰写新闻稿
4. 能根据提供的数据生成规范的报表

工作方式：
- 先确认公文类型和关键信息
- 按照学校标准格式生成
- 提供可修改的建议
- 语言正式、得体、简洁

注意事项：
- 涉及人事和财务信息时提醒人工审核
- 所有公文需经领导审批后发布
- 引用政策法规时注明出处
```

## 回复逻辑设计

### 1. 公文撰写流程

<div class="case-card">
<span class="case-label">📝 公文撰写对话流</span>

```
第1步：确认公文类型
  → "请问您需要什么类型的公文？"
  → 选项：通知/报告/总结/请示/新闻稿/其他

第2步：收集关键信息
  → 根据公文类型询问：
    - 通知：主题、对象、时间、地点
    - 报告：背景、目的、数据
    - 总结：时间段、主要工作、成果
    - 请示：事由、方案、预算

第3步：生成初稿
  → 按照标准格式生成
  → 标注需要人工确认的部分

第4步：修改完善
  → 根据用户反馈修改
  → 确认格式和内容
```

</div>

### 2. 回复风格设置

| 场景 | 风格 | 示例 |
|------|------|------|
| 公文撰写 | 正式、规范 | "根据XX文件精神，现将有关情况汇报如下..." |
| 会议纪要 | 条理清晰 | "会议讨论了以下三个议题：1...2...3..." |
| 新闻稿 | 生动、正面 | "近日，我校成功举办了XX活动..." |
| 数据汇总 | 客观、准确 | "本学期共开展XX次教研活动，参与教师XX人次..." |

## 知识库配置建议

| 文件 | 用途 | 推荐数量 |
|------|------|---------|
| 学校简介和基本信息 | 提供学校基础数据 | 1份 |
| 公文格式规范 | 确保格式正确 | 1-2份 |
| 往年优秀公文范例 | 参考风格 | 3-5份 |
| 政策法规摘要 | 常用政策引用 | 1份 |
| 学校管理制度 | 确保内容合规 | 1份 |
