## 细节优化建议

### 1. 评语温度设置

| 设置项 | 建议值 | 说明 |
|--------|--------|------|
| 语气风格 | 温暖鼓励型 | 避免负面评价 |
| 个性化程度 | 高 | 每个学生评语不同 |
| 字数控制 | 80-120字/人 | 简洁但有温度 |

### 2. 隐私保护配置

<div class="case-card">
<span class="case-label">🔒 隐私安全注意事项</span>

1. **不在知识库中存储学生个人信息**（姓名、家庭情况等）
2. **评语生成使用学号而非姓名**（保护隐私）
3. **家校沟通话术不包含敏感信息**（家庭背景、经济状况等）
4. **定期清理对话历史**（避免信息泄露）

</div>

### 3. 知识库配置

| 文件 | 用途 | 推荐数量 |
|------|------|---------|
| 学生评语词汇库 | 丰富评语表达 | 1份（200+条） |
| 家校沟通话术模板 | 提供话术参考 | 1份（30+条） |
| 班会方案范例 | 参考格式和内容 | 2-3份 |
| 班主任工作计划模板 | 计划撰写参考 | 1份 |

### 4. 插件配置

| 插件 | 用途 | 推荐度 |
|------|------|--------|
| 文档处理 | 批量导出评语文档 | ⭐⭐⭐⭐⭐ |
| 表格处理 | 整理班级数据 | ⭐⭐⭐⭐ |

## 多轮对话配置

| 设置项 | 建议值 | 说明 |
|--------|--------|------|
| 对话轮数 | 15-20轮 | 评语修改和家校沟通需要交互 |
| 开启对话记忆 | ✅ 开启 | 支持方案逐步完善 |
