# 8.3 行政助手插件的选择与配置

## 行政助手需要的插件

行政工作有其特殊性——需要引用最新政策、处理长文档、统计数据。选择合适的插件能大幅提升行政助手的实用性。

<div class="case-card">
<span class="case-label">📌 对比案例</span>

### 有插件 vs 无插件的差异

**无插件的行政助手**：
- 写通知时无法查询最新政策，只能凭训练数据
- 处理会议纪要时需要手动粘贴全部文字
- 数据统计需要人工计算

**有插件的行政助手**：
- 联网搜索最新政策文件，准确引用
- 文档阅读插件直接读取上传的会议录音转写文本
- 代码执行插件自动统计数据

> 💡 **插件不是越多越好，行政助手最核心的插件只有3个：联网搜索、文档阅读、代码执行。**

</div>

| 插件 | 用途 | 重要程度 | 行政场景示例 | 配置优先级 |
|------|------|---------|-------------|-----------|
| **联网搜索** | 查询最新政策、时事、法规 | ⭐⭐⭐⭐⭐ | 写公文时引用最新"双减"政策要求 | P0-必配 |
| **文档阅读** | 读取上传的文档进行整理 | ⭐⭐⭐⭐ | 读取会议录音转写文本后提取要点 | P0-必配 |
| **代码执行** | 数据统计与计算 | ⭐⭐⭐ | 成绩统计、考勤数据分析 | P1-推荐 |
| **日历** | 会议安排、日程管理 | ⭐⭐ | 自动查询日期和节假日信息 | P2-可选 |
| **邮件** | 自动发送通知和提醒 | ⭐⭐ | 群发会议通知 | P2-可选 |

## 📋 实操步骤

<div class="step-card">
<span class="step-number">1</span>
<span class="step-title">添加联网搜索插件</span>

这是行政助手最重要的插件——写公文时经常需要引用最新政策。

**操作步骤**：
1. 在智能体编辑页面，找到"插件"或"能力"配置区域
2. 在插件市场搜索"联网搜索"或"搜索"
3. 添加搜索结果插件（推荐选择覆盖面广的搜索引擎）
4. 测试：在预览窗口输入"查询最新的义务教育课程方案要求"
5. 确认AI能搜索并引用相关政策文件

**测试用例**：
```
输入：请查询2025年最新的"双减"政策要求，并帮我写一份学校落实"双减"的工作计划。
预期输出：AI先搜索相关政策文件，引用具体条文，然后生成工作计划。
```

**注意事项**：
- 联网搜索的结果需要人工核实，AI可能引用不准确
- 在人设中添加约束："引用政策时标注文件名称和文号，并提醒人工核实"

<span class="step-number">2</span>
<span class="step-title">添加文档阅读插件</span>

用于读取上传的长文档后提取要点。典型场景：会议录音转写文本、工作总结素材、政策文件等。

**操作步骤**：
1. 在插件市场搜索"文档阅读"或"文件处理"
2. 添加文档阅读插件
3. 测试：上传一份会议录音转写文本（约3000字）
4. 输入"请阅读这份文件，提取会议讨论的主要议题和达成的共识"
5. 确认AI能准确提取要点

**测试用例**：
```
上传文件：会议纪要原文.txt（约3000字）
输入：请阅读这份文件，提取以下信息：
1. 会议讨论了哪些议题？
2. 达成了哪些共识？
3. 有哪些待办事项？
4. 各项待办的责任人是谁？
预期输出：结构化的会议纪要摘要。
```

<span class="step-number">3</span>
<span class="step-title">配置知识库</span>

知识库是行政助手的"内部资料库"，上传学校相关文件后，AI可以精准检索。

**推荐上传的知识库文件**：

| 文件类型 | 内容 | 推荐数量 | 示例 |
|----------|------|---------|------|
| 学校简介 | 学校基本信息（人数、班级数、校训等） | 1份 | XX学校简介2025版 |
| 公文格式规范 | 学校公文标准格式 | 1-2份 | XX学校公文格式规范 |
| 往年优秀公文 | 参考风格和格式 | 3-5份 | 2024年优秀公文汇编 |
| 政策法规摘要 | 常用政策要点 | 1份 | 教育政策法规摘要 |
| 学校管理制度 | 确保内容合规 | 1份 | XX学校管理制度汇编 |

**知识库配置步骤**：
1. 在智能体编辑页面找到"知识库"配置区域
2. 点击"上传文件"，逐个上传上述文件
3. 等待文件索引完成（通常需要1-3分钟）
4. 测试：输入"我们学校的校训是什么"，确认AI能从知识库中检索
</div>

## 插件冲突处理

<div class="case-card">
<span class="case-label">⚠️ 插件使用注意事项</span>

### 常见问题与解决

**问题一：联网搜索和知识库结果冲突怎么办？**

当联网搜索的结果与知识库内容不一致时（如政策已更新但知识库还是旧版本）：
- 在人设中设置优先级："优先使用知识库内容，如联网搜索结果与知识库不一致，以知识库为准，并提醒用户核实最新版本"

**问题二：插件调用失败怎么办？**

- 联网搜索可能因网络问题失败 → 人设中添加："如果联网搜索失败，请告知用户并基于已有知识回答，同时建议稍后重新搜索核实"
- 文档阅读可能因文件格式不支持失败 → 建议使用PDF或TXT格式

**问题三：权限管理**

- 行政助手可能涉及敏感信息 → 建议在团队空间中设置访问权限，仅限行政人员使用
- 知识库中不要上传涉密文件（人事变动、财务数据等）

</div>

## 使用技巧

### 技巧一：引用最新政策

```
请帮我搜索2025年最新的"双减"政策要求，
然后写一份学校落实"双减"的工作总结。
要求：
1. 引用政策文件时标注文件名称
2. 结合我校实际情况
3. 包含已完成的工作和下一步计划
```

### 技巧二：读取文件后整理

```
请阅读我上传的各部门工作总结，
汇总成一份学校年度工作总结，
要求：
1. 有数据支撑
2. 突出亮点和成果
3. 客观分析不足
4. 提出下学期展望
```

### 技巧三：数据统计

```
请帮我分析以下考勤数据：
- 教师总人数：85人
- 本月出勤率：96.5%
- 缺勤原因分布：病假40%、事假35%、其他25%
- 去年同期出勤率：94.2%

请生成一份考勤分析报告，包含：
1. 总体情况
2. 与去年同期对比
3. 缺勤原因分析
4. 改进建议
```

## 测试检查清单

| 测试项 | 方法 | 通过标准 |
|--------|------|---------|
| 联网搜索 | 搜索最新政策 | 能返回相关政策信息 |
| 文档阅读 | 上传3000字文档提取要点 | 要点提取准确率≥80% |
| 知识库检索 | 查询学校基本信息 | 检索准确，无张冠李戴 |
| 插件协同 | 联网搜索+知识库组合使用 | 结果一致或有不一致时能说明 |
