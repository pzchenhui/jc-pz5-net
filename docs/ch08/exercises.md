# 📝 第8章练习：班主任的AI分身

## 练习目标

搭建班主任助手智能体，能生成学生评语、记录家校沟通、设计班会方案。

---

## 🛠️ 实操任务：搭建班主任助手

### 步骤检查清单

| 步骤 | 完成标准 | 打勾 |
|------|---------|------|
| 1. 编写班主任人设 | 温暖、有原则 | □ |
| 2. 创建评语生成对话流 | 收集学生信息→生成评语 | □ |
| 3. 测试评语生成 | 评语有温度、个性化 | □ |
| 4. 测试班会设计 | 方案有教育意义 | □ |
| 5. 测试家校沟通 | 建议专业、温和 | □ |
| 6. 发布试用 | 分享给班主任同事 | □ |

---

## 📋 练习二：对话流设计（20分钟）

**设计一个"学生情况登记"对话流：**

```
开始
  ↓
节点1：[填写] _______________
  ↓
节点2：[填写] _______________
  ↓
节点3：[填写] _______________
  ↓
节点4：生成档案摘要
  ↓
结束
```
